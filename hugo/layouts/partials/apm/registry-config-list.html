Only shows current entries (to_version is null), sorted by name within each category. Each section and config entry has an anchor ID with a copy-link button. */}} {{- $lang := "" -}} {{- with .Page -}} {{- $lang = .Params.code_lang -}} {{- end -}} {{- $data := index site.Data.registry $lang -}} {{- $categories := site.Data.registry.categories -}} {{- if $data -}} {{- $configs := $data.configurations -}} {{/* Split configurations into active and retired. - Active: to_version is null, "latest", or empty. - Retired: to_version is a specific version (e.g. "v1.57.0"); the key was supported in versions strictly less than that. A retired entry is only shown if no active entry exists for the same key name — otherwise it's just a prior version of a still-current key, which would mislead readers into thinking the key was removed.
