title: test de aplicaciones móviles
---
En la monitorización Synthetic de Datadog, un test de aplicaciones móviles monitoriza los flujos comerciales clave que pueden implicar varias acciones y páginas, para verificar que los usuarios pueden completar con éxito procesos como registrarse para tener una cuenta y realizar un pago. Para obtener más información, <a href="/mobile_app_testing/">consulta la documentación</a>.
