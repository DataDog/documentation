<div class="alert alert-info">Datadog does not send security notifications through webhooks due to HIPAA restrictions. Security alerts won't be sent to the webhook for HIPAA-enabled accounts. If you have a HIPAA-enabled account, you cannot use <code>@webhook...</code> in the <b>Notify the following recipients</b> setting within Datadog security notifications. If you want these alerts sent, please <a href="https://docs.datadoghq.com/help/">contact support</a>.</div>
