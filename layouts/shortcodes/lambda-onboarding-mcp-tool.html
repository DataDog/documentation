Use the Datadog MCP Server's serverless_onboarding tool with an AI coding assistant to instrument your Lambda functions. The tool analyzes your project and applies the necessary configuration changes.

  1. Set up the Datadog MCP Server in your AI coding assistant (Cursor, Claude Code, or another MCP-compatible client). Include the core and onboarding toolsets by appending ?toolsets=core,onboarding to the endpoint URL.

  2. Paste the following prompt into your AI agent:

    Instrument my AWS Lambda functions with Datadog
