[Datadog CLI ツール][102]
: `datadog-ci` クライアントのバージョン 2.4.1 以降を使用してください。CLI ツールはコードリポジトリと同じディレクトリで実行する必要があります。

[Datadog Serverless Plugin][103]
: プラグインのバージョンは 5.18.0 以降を使用してください。

[Datadog CDK Construct][104]
: AWS CDK v1 では `datadog-cdk-constructs` のバージョン 0.8.5 以降を使用してください。</br> AWS CDK v2 では `datadog-cdk-constructs` のバージョン 1.4.0 以降を使用してください。
