Lorsque la solution Test Visibility est activée, les données suivantes sont recueillies à partir de votre projet :

* Les noms et durées des tests
* Les variables d'environnement prédéfinies par les fournisseurs de CI
* L'historique de commits Git, y compris les informations sur les hashs, les messages et les auteurs, ainsi que sur les modifications de fichier (sans le contenu des fichiers)
* Les informations provenant du fichier CODEOWNERS
