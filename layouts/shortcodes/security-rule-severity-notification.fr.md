Dans le menu déroulant **Set severity to**, sélectionnez le niveau de gravité approprié (`INFO`, `LOW`, `MEDIUM`, `HIGH`, `CRITICAL`).

Dans la section **Notify**, vous pouvez configurer des [cibles de notification][101] pour chaque cas de règle.

Il est également possible de créer des [règles de notification][102], afin dʼéviter de modifier des préférences de notification pour chaque règle de détection.
