If you do not have a Datadog Log Archive configured for Observability Pipelines, configure a Log Archive for your cloud provider ([Amazon S3][1041], [Google Cloud Storage][1042], or [Azure Storage][1042]).

**Note**: You need to have the Datadog integration for your cloud provider installed to set up Datadog Log Archives. See [AWS integration][1043], [Google Cloud Platform][1043], and [Azure integration][1044] documentation for more information.
