If you do not have a Datadog Log Archive configured for Observability Pipelines, configure a Log Archive for your cloud provider ([Amazon S3][10041], [Google Cloud Storage][10042], or [Azure Storage][10043]).

**Note**: You need to have the Datadog integration for your cloud provider installed to set up Datadog Log Archives. See the [AWS integration][10044], [Google Cloud Platform][10045], and [Azure integration][10046] documentation for more information.
