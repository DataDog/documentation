Optionally, check the box to enable TLS. If you enable TLS, the following certificate and key files are required:
- `Server Certificate Path`: The path to the certificate file that has been signed by your Certificate Authority (CA) Root File in either DER, PEM (X.509).
- `CA Certificate Path`: The path to the certificate file that is your Certificate Authority (CA) Root File in either DER, PEM (X.509).
- `Private Key Path`: The path to the `.key` private key file that belongs to your Server Certificate Path in either DER or PEM (PKCS#8) format.
