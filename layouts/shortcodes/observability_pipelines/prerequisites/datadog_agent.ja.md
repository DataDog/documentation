Datadog Agent は、ログを収集して [Datadog Log Management][2001] にルーティングするためにすでにインストールされています。Datadog Agent をセットアップしていない場合は、[Datadog Agent ドキュメント][2002]を参照してください。

次の情報が利用可能です。
- [リモートコンフィギュレーションが有効][2003]な Datadog API キー。
- Datadog [`Site URL`][2004]。たとえば、サイト `US1` の場合は `datadoghq.com` です。
