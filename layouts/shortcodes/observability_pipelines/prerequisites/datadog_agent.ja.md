ログを収集し、[Datadog Log Management][2001] にルーティングするための Datadog Agent はすでにインストールされています。Datadog Agent がセットアップされていない場合は、[Datadog Agent ドキュメント][2002]を参照してください。

次の情報が利用可能です。
- [リモート構成が有効][2003]な Datadog API キー。
- Datadog の [`Site URL`][2004]。たとえば、`US1` サイトの場合は `datadoghq.com`。
