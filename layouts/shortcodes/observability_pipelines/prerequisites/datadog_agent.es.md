Ya tienes instalado el Datadog Agent para recopilar y dirigir tus logs a [Datadog Log Management][2001]. Si no tienes Datadog Agents instalados, consulta la [documentación del Datadog Agent][2002] para más información.

Dispones de la siguiente información:
- Una clave de API de Datadog con la [Configuración remota activada][2003].
- Tu [`Site URL`][2004] de Datadog. Por ejemplo, `datadoghq.com` para el sitio `US1`.
