Ya tienes instalado el Datadog Agent para recoger y dirigir tus logs a [Datadog Log Management][2001]. Si no tienes configurados los Datadog Agent, consulta la [documentación del Datadog Agent][2002] para obtener más información.

Dispones de la siguiente información:
- Una clave de API de Datadog con la [Configuración remota activada][2003].
- Tu Datadog [`Site URL`][2004]. Por ejemplo, `datadoghq.com` para el sitio `US1`.
