# Documentation Review Skill

When invoked, re-read the target file fresh, then review for:
1. **Parallelism**: Ensure list items and headings use consistent grammatical structure
2. **Passive voice**: Flag every instance of passive voice and suggest active alternatives
3. **Grammar & clarity**: Fix grammar issues and simplify complex sentences
4. **Flow**: Check logical ordering of sections and smooth transitions
5. **Style guide compliance**: Follow Datadog documentation style conventions
6. **Cross-links**: Verify internal links are present where relevant topics are mentioned

Present findings grouped by category with line numbers.
