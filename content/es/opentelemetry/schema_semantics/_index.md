## Información general

OpenTelemetry utiliza convenciones semánticas para especificar los nombres de los diferentes tipos de datos. Las convenciones semánticas definen nombres y valores comunes para los datos de telemetría con el fin de promover la coherencia entre los sistemas.

En la siguiente documentación, se describe cómo se corresponden las convenciones de OpenTelemetry y Datadog.
