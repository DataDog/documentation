## Información general

Instrumentar tu aplicación y habilitar ASM te permite obtener rápidamente información de seguridad de una amplia gama de operaciones estándar y marcos populares con una intervención manual mínima. Puedes Instrumentar automáticamente tu aplicación cuando instalas el Datadog Agent con [la instrumentación de un solo paso][1] o cuando añades manualmente bibliotecas de rastreo de Datadog a tu código.

La siguiente documentación proporciona instrucciones para habilitar ASM utilizando las bibliotecas de rastreo de Datadog para la detección de amenazas y las capacidades de protección, y para la seguridad del código.
