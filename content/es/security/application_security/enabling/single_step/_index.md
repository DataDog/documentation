## Información general

Instrumentar tu aplicación y habilitar ASM te permite obtener rápidamente información de seguridad de una amplia gama de operaciones estándar y marcos populares con una intervención manual mínima. Puedes Instrumentar automáticamente tu aplicación cuando instalas el Datadog Agent con la instrumentación de un solo paso o cuando [añades manualmente las bibliotecas de rastreo de Datadog][1] al código.

La siguiente documentación proporciona instrucciones para habilitar ASM con la instrumentación de un solo paso para las capacidades de detección y protección ante amenazas, y para la seguridad del código.
