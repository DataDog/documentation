Las siguientes funciones son compatibles con respecto a la librería de rastreo de cada lenguaje:

| Función de Code Security                        | Java    | .NET     | Node.js                                          | Python        | Go              | Ruby          | PHP           |
|------------------------------------------------|---------|----------|--------------------------------------------------|---------------|-----------------|---------------|---------------|
| Runtime Software Composition Analysis (SCA)    | 1.1.4   | 2.16.0   | 4.0.0                                            | 1.5.0         | 1.49.0          | 1.11.0        | 0.90.0        |

**Nota**: Las funciones de **Static Software Composition Analysis (SCA)** y **Static Code Analysis (SAST)** no requieren la librería de rastreo de Datadog. Por lo tanto, los requisitos mencionados a continuación no se aplican a estas dos funciones de Code Security.

Selecciona el lenguaje de tu aplicación para obtener más información sobre la compatibilidad de marcos y funciones.
