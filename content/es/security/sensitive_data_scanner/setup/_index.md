## Información general

Configura Sensitive Data Scanner para escanear tus:

- Datos de telemetría, para que puedas identificar datos confidenciales en tus logs, tramos de APM, eventos de RUM y eventos de Event Management. Consulta [Configuración para datos de telemetría][1] para obtener instrucciones.
- Datos de almacenamiento en la nube, para que puedas identificar datos confidenciales en tus buckets de Amazon S3 e instancias de RDS. Consulta [Configurar para Cloud Storage][2] para obtener instrucciones.
