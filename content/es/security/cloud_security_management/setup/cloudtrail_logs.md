---
title: Configuración de AWS CloudTrail Logs para Cloud Security
---

Configura AWS CloudTrail Logs para sacar el máximo partido de [Cloud Security Identity Risks][1]. AWS CloudTrail Logs proporciona información adicional sobre el uso real de los recursos en la nube, ayudándote a identificar a los usuarios y funciones con diferencias significativas entre los permisos concedidos y los utilizados.
