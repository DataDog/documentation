title: gráfica dividida
---
Una gráfica dividida permite desglosar una consulta en varios valores de etiqueta (tag) para identificar outliers y patrones. 

Utiliza esta función para investigar el rendimiento de las métricas en varias facetas, comparar eventos en diferentes etiquetas o crear visualizaciones dinámicas. Para obtener más información, <a href="/dashboards/widgets/split_graph/">consulta la documentación</a>.
