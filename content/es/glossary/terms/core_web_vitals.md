title: Core Web Vitals
---
Core Web Vitals es un conjunto de métricas que Google ha identificado como señales significativas para las pruebas de experiencia del usuario. Estas métricas se centran en cuánto tarda en cargarse una página, cuánto tarda un usuario en interactuar, qué tan estable es una página mientras se carga, etc. Para obtener más información, <a href="https://support.google.com/webmasters/answer/9205520?hl=en/">consulta la documentación</a>.
