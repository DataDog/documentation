title: resumen de servicio
---
Un servicio es un conjunto de procesos que hacen el mismo trabajo, por ejemplo, un marco web o una base de datos. Datadog proporciona gráficas listas para utilizar a fin de mostrar información del servicio, como se ve en la <a href="/tracing/services/service_page/">Página de servicios</a>. 

Utiliza el widget de resumen de servicio para visualizar las gráficas de un servicio elegido en tu dashboard. Para obtener más información, <a href="/dashboards/widgets/service_summary/">consulta la documentación</a>.
