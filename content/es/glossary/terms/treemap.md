title: treemap
---
El widget de treemap permite visualizar proporciones de uno o varios conjuntos de datos. 

Este widget puede mostrar un único conjunto de datos con las proporciones correspondientes, o varios conjuntos de datos con proporciones anidadas. Para obtener más información, <a href="/dashboards/widgets/treemap/">consulta la documentación</a>.
