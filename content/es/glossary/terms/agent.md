El Datadog Agent es un software de código abierto que se ejecuta en un host. Recopila métricas y eventos del host y los envía a Datadog. Se puede ejecutar en hosts locales (Windows, macOS), en entornos contenedorizados (Docker, Kubernetes) y en centros de datos locales.
Para obtener más información, <a href="/getting_started/agent/">consulta la documentación</a>.
