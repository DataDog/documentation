title: valor de consulta
---
Los valores de consulta muestran el valor actual de una métrica, APM o consulta de log determinada. Cuentan con un formato condicional (como un fondo verde/amarillo/rojo) para indicar si el valor está en el rango esperado. Esto se puede complementar con fondos opcionales de datos de series temporales. 

Los valores que muestra un valor de consulta no requieren una medición instantánea. El widget puede mostrar el último valor informado o un agregado calculado a partir de todos los valores de consulta durante el intervalo de tiempo. Para obtener más información, <a href="/dashboards/widgets/query_value/">consulta la documentación</a>.
