title: mapa geográfico
---
El widget de mapa geográfico visualiza datos geográficos con regiones o puntos sombreados. Utiliza este widget para ver las sesiones de usuario por país, filtrar a fin de consultar una lista de todas las sesiones en una pestaña nueva, o monitorizar métricas de rendimiento como el tiempo de carga, Core Web Vitals y el porcentaje de vistas con errores. Para obtener más información, <a href="/dashboards/widgets/geomap/">consulta la documentación</a>.
