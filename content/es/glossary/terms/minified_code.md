title: código minificado
---
El código minificado es el código (a menudo JavaScript) despojado de comentarios, espacios en blanco adicionales, código sin utilizar y cualquier otra cosa que no afecta a la funcionalidad. El código minificado es menos legible, pero su menor tamaño mejora el rendimiento del sitio web.
