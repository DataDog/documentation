title: DogStatsD
---
DogStatsD hace referencia a dos cosas relacionadas: un protocolo basado en StatsD, y una aplicación para informar métricas que implementa dicho protocolo. El protocolo de DogStatsD es una extensión del protocolo de StatsD, con algunas modificaciones propias de la plataforma de Datadog. La aplicación de DogStatsD es un servicio que se incluye con el Agent y se utiliza como mecanismo ligero para informar métricas.

Para obtener más información, consulta la <a href='/developers/dogstatsd'>documentación de DogStatsD</a>.
