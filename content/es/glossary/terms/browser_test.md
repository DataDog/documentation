title: test de navegador
---
En la monitorización Synthetic de Datadog, un test de navegador monitoriza las transacciones comerciales web o los recorridos del usuario web. Un solo test puede incluir varias acciones y páginas para verificar que los usuarios puedan completar con éxito procesos como registrarse a fin de obtener una cuenta y realizar un pago. Para obtener más información, <a href="/synthetics/browser_tests/">consulta la documentación</a>.
