## Información general

Datazoom es una plataforma de datos de vídeo que recopila datos de endpoints a través de un ecosistema de recopiladores.

El [Datazoom Datadog Connector](https://help.datazoom.io/hc/en-us/articles/360042494512-Datadog) envía los datos del recopilador a Datadog, donde puedes consultar los datos en tu [Log Explorer](https://app.datadoghq.com/logs).

Datazoom envía datos configurados en el nivel INFO.

## Configuración

### Instalación

La integración Datazoom emite logs a Datadog. No es necesaria ninguna instalación del lado de Datadog.

### Configuración

- Visita la [documentación] de integración de Datazoom (https://help.datazoom.io/hc/en-us/articles/360042494512-Datadog) para obtener información detallada sobre cómo configurar el Datazoom Datadog Connector.

### Dashboard

Consulta el [dashboard de logs de Datazoom](https://app.datadoghq.com/dashboard/lists/preset/3?q=datazoom).

## Datos recopilados

### Métricas

Datazoom no incluye métricas.

### Checks de servicio

Datazoom no incluye checks de servicio.

### Eventos

Datazoom no incluye eventos.
