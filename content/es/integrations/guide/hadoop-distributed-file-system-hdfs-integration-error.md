Si encuentras errores al integrar HDFS, primero intenta actualizar pip y Snakebite siguiendo estos pasos:

1. Ejecuta: /opt/datadog-agent/embedded/bin/pip install --upgrade pip
2. Ejecuta: /opt/datadog-agent/embedded/bin/pip install --upgrade snakebite
3. [Reinicia el Agent][1].

Si estos pasos no resuelven el error, ponte en contacto con el [servicio de asistencia de Datadog][2].
