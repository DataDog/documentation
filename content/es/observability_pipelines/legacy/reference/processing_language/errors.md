Datadog Processing Language (DPL) o Vector Remap Language (VRL), es un lenguaje a prueba de fallos, lo que significa que un programa DPL/VRL no compila a menos que se manejen todos los errores potenciales. Esto asegura que tus programas DPL/VRL puedan manejar datos malformados.
