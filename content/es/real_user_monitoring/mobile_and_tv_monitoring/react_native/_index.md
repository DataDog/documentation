## Información general

Datadog Real User Monitoring (RUM) te permite visualizar y analizar el rendimiento en tiempo real y los recorridos de cada usuario de tu aplicación.

## Comenzar a monitorizar aplicaciones React Native

Para empezar con RUM para React Native, crea una aplicación y configura el SDK de React Native.
