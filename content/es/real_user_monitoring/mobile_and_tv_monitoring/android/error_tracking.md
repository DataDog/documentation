## Información general

Errores en los procesos de seguimiento de errores recopilados del SDK de Android.

Habilita la notificación de fallos y el seguimiento de errores Android para obtener informes completos de fallos y tendencias de errores. Con esta función, puedes acceder a:

- Dashboards y atributos de fallos agregados Android
- Informes de fallos de Android desenmascarados con asignación de stack trace
- Informes de fallos del NDK para código nativo (C/C++)
- Informe de ANR (Aplicación no responde)
- Análisis de tendencias con el seguimiento de errores Android

Tus informes de bloqueos aparecen en [**Error Tracking**][1].

## Configuración

Si aún no has configurado el SDK para Android, sigue las [instrucciones de configuración en la aplicación][2] o consulta la [documentación de configuración de Informe de fallos y Error Tracking de Android][3].
