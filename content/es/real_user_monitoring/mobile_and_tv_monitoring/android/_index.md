## Información general

Datadog Real User Monitoring (RUM) te permite visualizar y analizar el rendimiento en tiempo real y los recorridos de cada usuario de tu aplicación.

El SDK de Datadog Android es compatible con Android 5.0+ (API nivel 21) y Android TV. Funciona con aplicaciones Android escritas en Java o Kotlin.

## Iniciar la monitorización de aplicaciones de Android

Para empezar con RUM para Android, crea una aplicación y configura el SDK de Android.
