## Información general

Datadog Real User Monitoring (RUM) te permite visualizar y analizar el rendimiento en tiempo real y los recorridos de cada usuario de tu aplicación.

## Iniciar la monitorización de aplicaciones de Unity

Para empezar con RUM para Unity, crea una aplicación y configurar el SDK de Unity.
