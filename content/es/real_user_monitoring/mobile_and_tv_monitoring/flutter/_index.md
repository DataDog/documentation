## Información general

Datadog Real User Monitoring (RUM) te permite visualizar y analizar el rendimiento en tiempo real y los recorridos de cada usuario de tu aplicación.

## Empezar a monitorizar aplicaciones Flutter

Para empezar a utilizar RUM para Flutter, crea una aplicación y configura el SDK Flutter.
