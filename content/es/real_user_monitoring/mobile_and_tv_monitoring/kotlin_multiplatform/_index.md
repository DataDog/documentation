## Información general

Datadog Real User Monitoring (RUM) te permite visualizar y analizar el rendimiento en tiempo real y los recorridos de cada usuario de tu aplicación.

El SDK de Datadog Kotlin Multiplatform es compatible con Android 5.0+ (nivel de API 21) e iOS v12+.

## Iniciar monitorización de aplicaciones de Kotlin Multiplatform

Para empezar con RUM para Kotlin Multiplatform, crea una aplicación y configura el SDK de Kotlin Multiplatform.
