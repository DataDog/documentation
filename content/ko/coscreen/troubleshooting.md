### 블루투스 헤드셋의 마이크를 CoScreen, Zoom 및 기타 툴의 입력값으로 사용할 때 오디오 품질이 저하되는 이유는 무엇인가요?

블루투스 헤드셋을 사용하는 경우 헤드셋의 마이크를 오디오 입력 장치로 선택하면 플레이백 품질이 저하될 수 있습니다. CoScreen 세션 중에 오디오(예: YouTube 비디오)를 재생하는 경우 이러한 현상이 나타날 수 있습니다. 이는 블루투스 헤드셋이 다른 블루투스 프로파일을 사용하도록 전환되었기 때문에 발생할 수 있습니다.

오디오만 재생할 때 블루투스 헤드셋은 보통 [A2DP 프로파일][2]을 사용하는데, 이는 고음질에 최적화되어 있지만 마이크 사용을 지원하지 않습니다. 헤드셋 마이크를 오디오 입력값으로 선택하면(예: CoScreen 세션 또는 Zoom 회의 중) 헤드셋은 보통 마이크 사용을 지원하지만 음질이 낮은 다른 프로파일인 [HFP][3] 또는 [HSP][4]로 전환됩니다. 대부분의 블루투스 헤드셋은 한 번에 하나의 프로파일만 사용할 수 있습니다.

이러한 문제를 방지하려면 노트북의 내장 마이크와 같은 다른 오디오 입력값을 사용할 수 있습니다. 애플리케이션을 재시작해야 고품질 오디오가 다시 출력됩니다.

### 화면 공유 품질과 원격 제어 레이턴시를 최적화하려면 어떻게 해야 하나요?

사용자와 동료가 공유하는 Windows의 품질에 영향을 미치는 몇 가지 중요 요소가 있습니다.

#### 네트워크 대역폭 및 안정성

* 모든 참가자가 최소 5Mbps 업로드 및 다운로드 속도를 갖춘 빠르고 안정적인 인터넷에 연결되어 있는지 확인하세요.
* 시스템 패킷 손실이 2%를 넘지 않는지 확인합니다. 공급자에 관계없이 해당 임계값을 넘어서게 되면 비디오 회의는 불안정해집니다. [패킷 손실 테스트][1]를 실행하세요. 문제가 발생하면 라우터 또는 모뎀을 다시 시작하거나, 라우터에 더 가까이 이동하거나 다른 인터넷 연결로 테스트해 보세요.
* CoScreen은 두 명의 참가자만 P2P로 연결할 수 있는 경우(즉, 기업 프록시 또는 방화벽을 거치지 않고 직접 연결할 수 있는 경우) 가장 빠르게 작동합니다. 참가자가 3명 이상이거나 직접 연결이 불가능하면 전 세계에 분산된 비디오 인프라스트럭처를 통해 트래픽을 라우팅하여 어떠한 상황에서도 연결성을 보장합니다.

#### 화면 해상도
CoScreen은 사용자와 동료가 4K 이상의 초고해상도 디스플레이를 사용하는 경우에도 고해상도 화면 공유를 지원합니다. 네트워크 대역폭과 CPU가 해당 부하를 처리할 수 있는지 확인하세요. 문제가 발생한 경우 양측에서 더 낮은 해상도로 시도해 보세요.

#### CPU
CoScreen은 다른 많은 툴보다 훨씬 높은 품질로 캡처하므로 Windows 공유가 구형 시스템에서는 CPU 코어의 최대 60%를 차지할 수 있습니다. CPU 할당을 점검하세요.

### 방화벽 및 SSL
조직에서 회사 방화벽 또는 SSL 검사를 사용하는 경우 CoScreen 클라이언트가 CoScreen 서버에 연결할 수 없으므로 사용자와 팀원 간에 연결을 설정할 수 없습니다. 허용 목록에 추가할 URL 목록에 대해서는 지원 팀에 문의하세요.

### macOS에서 트러블슈팅

#### 측면 패널 UI가 회색으로 표시되고 연결이 설정되지 않습니다.

UI가 회색으로 표시되고 참여 단계에서 멈춰 있지만 'CoScreen에 연결할 수 없습니다' 대화 상자가 표시되지 않는 문제가 발생할 수 있습니다. CoScreen은 비고정 단일 데스크톱에서 실행될 것으로 예상하기 때문입니다. 해당 문제는 특히 애플리케이션이 두 개 이상의 데스크톱에 있도록 설정한 경우 발생할 수 있습니다.

이 문제를 해결하려면 다음을 따르세요.

1. CoScreen 애플리케이션 아이콘을 마우스 오른쪽 버튼으로 클릭하고 _Options_으로 이동합니다.
2. _Assign To_이 _None_으로 설정되어 있는지 확인합니다.
3. _Options_을 종료하고 CoScreen에 다시 참여합니다.

### Windows 트러블슈팅

#### CoScreen은 Windows 관리자 모드에서 어떻게 작동하나요?

관리자 모드에서 CoScreen을 실행하고, 관리자 모드에서 실행된 앱을 공유하는 경우 모든 원격 사용자가 공유 창을 제어할 수 있습니다. 그러나 관리자 모드가 아닌 상태에서 CoScreen을 실행하고, 관리자 모드에서 실행된 앱을 공유하는 경우 원격 사용자는 해당 창을 제어할 수 없습니다.

#### CoScreen UI가 작게 표시됩니다.

CoScreen UI가 Windows 컴퓨터의 다른 앱보다 작게 표시되는 경우, 이는 디스플레이 설정에서 설정한 비율과 CoScreen과의 상호 작용 방식과 연관되어 있습니다. 화면의 비율과 해상도를 낮춰서 사용 환경을 개선할 수 있습니다.

#### 목소리가 잘 들리지 않거나 로봇처럼 들립니다.

일부 기기는 Realtek 음성 인식 기능을 사용하기 때문에 CoScreen을 사용할 때 목소리가 로봇처럼 들릴 수 있습니다. 음성이 잘 들리지 않는 문제가 발생하면 Realtek 오디오 콘솔을 열고 **Voice Recognition** 옵션을 해제하여 상황이 개선되는지 확인하세요.
