## 개요

Watchdog 경고 기능은 서비스 및 인프라스트럭처 이상을 사전에 감지합니다. Watchdog은 모니터링되는 모든 기술에서 비정상적인 동작을 검색합니다. 비정상적인 동작이 Watchdog의 이상 패턴 중 하나와 일치하면 Watchdog에서 경고를 생성합니다.

## 경고

### Watchdog 경고를 찾을 수 있는 위치

Watchdog 경고는 Datadog 사이트 내의 다음 세 위치에 나타납니다.
- [Watchdog 경고 피드][1]
- [APM 홈페이지][2]
- 개별 [APM 서비스 페이지][3]

### 경고 개요

각 경고 개요 카드는 하나의 이상에 대한 정보를 전달합니다.

{{< img src="watchdog/alerts/alerts_overview.png" alt="sms-service의 send-sms 엔드포인트에서 높은 오류율을 보여주는 Watchdog 경고 카드의 스크린샷" style="width:100%;">}}

경고 개요 카드에는 다음 섹션이 포함되어 있습니다.

1. 상태: 이상의 상태는 **진행 중** 또는 **해결됨**일 수 있습니다.
2. 타임라인: 이상이 발생한 기간을 설명합니다.
3. 메시지: 이상을 설명합니다.
4. 그래프: 이상을 시각적으로 나타냅니다.
5. 태그: 이상의 범위를 표시합니다.
6. [영향][4](해당되는 경우): 이상이 영향을 미치는 사용자, 보기 또는 서비스를 설명합니다.

관련 없는 경고를 숨기려면 해당 경고 위로 마우스를 가져갑니다. 오른쪽 상단 모서리에 나타나는 폴더 아이콘을 클릭하세요.

### 경고 상세 정보

경고 개요 카드의 아무 곳이나 클릭하면 경고 상세 정보 창이 열립니다.

경고 개요 카드의 정보를 반복하는 것 외에도 **Overview** 탭에는 다음 필드 중 하나 이상이 포함될 수 있습니다.
- 예상 범위: **Show expected bounds** 확인란을 클릭합니다. 예상 동작과 비정상 동작을 구분하기 위해 그래프 색상이 변경됩니다.
- 제안된 다음 단계: 비정상적인 동작을 조사하고 분류하기 위한 단계를 설명합니다.
- 상관 대시보드: 경고와 관련된 일부 대시보드를 제안합니다. Datadog은 경고의 인사이트와 관련된 대시보드의 메트릭을 강조 표시합니다.

**Monitors** 탭에는 경고와 관련된 모니터가 나열됩니다. 표시된 각 모니터에는 현재 경고의 메트릭과 해당 범위에 포함된 관련 태그가 있습니다.

또한 Watchdog은 이상이 다시 발생하면 그 사실을 알리기 위해 사용자에게 하나 이상의 모니터를 생성할 것을 제안합니다. 이러한 모니터는 아직 존재하지 않으므로 표에 상태가 **제안됨**으로 표시됩니다. 조직에 제안된 모니터를 활성화하려면 **Enable Monitor**를 클릭합니다. 새 모니터를 열거나, 편집하거나, 복제하거나, 음소거하거나, 삭제할 수 있는 일련의 아이콘이 나타납니다.

## 경고 필터링

시간 범위, 검색창 또는 패싯을 사용하여 Watchdog 경고 피드를 필터링할 수 있습니다.

### 시간 범위

특정 시간 범위에서 감지된 경보를 보려면 오른쪽 상단의 시간 범위 선택기를 사용하세요. 지난 13개월 동안 발생한 모든 경고를 볼 수 있습니다.

### 검색창

**Filter alerts** 검색 상자에 텍스트를 입력하여 경고 타이틀을 검색합니다.

### 패싯

Watchdog 경고 피드의 왼쪽에는 아래의 검색 패싯이 포함되어 있습니다. 패싯별로 경고를 필터링하려면 해당 상자를 선택하세요.

| 패싯           | 설명                                                                       |
|-----------------|-----------------------------------------------------------------------------------|
| 경고 범주  | 모든 `apm` 또는 모든 `infrastructure` 경고를 표시합니다.                                 |
| 경고 유형      | APM 또는 인프라스트럭처 통합의 메트릭을 사용하여 경고를 선택합니다.              |
| APM 환경 | 경고를 표시할 [APM 환경][5]입니다.                                  |
| APM 기본 태그 | 경고를 표시할 [정의된 APM 기본 태그][6]입니다.                          |
| APM 서비스     | 경고를 표시할 [APM 서비스][7]입니다.                                      |


## 보관된 경고 관리

Watchdog 경고의 사이드 패널에서 오른쪽 상단 모서리에 있는 폴더 아이콘을 클릭하여 해당 경고를 보관합니다. 보관을 하면 홈 페이지와 같은 Datadog 사이트의 다른 위치뿐만 아니라 피드에서 경고가 사라집니다. 경고가 보관된 경우 관련 서비스 또는 리소스 옆에 분홍색 Watchdog 쌍안경 아이콘이 표시되지 않습니다.

보관된 경고를 보려면 [Watchdog alerts feed][1]의 왼쪽 상단에 있는 "Show N archived alerts" 확인란 옵션을 선택합니다. 이 옵션은 보관된 경고가 하나 이상 있는 경우에만 사용할 수 있습니다. 또한 각 경고를 보관한 사람과 경고가 보관된 시기를 확인하고 보관된 경고를 피드에 복원할 수 있습니다.

**참고**: 보관을 해도 Watchdog이 서비스 또는 리소스와 관련된 향후 문제에 플래그를 지정하는 것이 금지되지 않습니다.
