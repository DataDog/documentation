## 개요

<div class="alert alert-info">실제 사용자 모니터링(RUM)이 있는 경우 RUM 미리보기, Session Replay, Error Tracking 및 Synthetic 브라우저 테스트의 추가 성능 데이터에 액세스할 수 있습니다.</div>

Synthetic 브라우저 테스트에는 Real User Monitoring SDK가 포함되어 있어 다음을 탐색할 수 있습니다.

- 모든 테스트에 대한 [Session Replay[1]. 각 단계와 [브라우저 개발 도구][2]에 사용 가능한 스크린샷 외에도 추가 컨텍스트에 액세스할 수 있습니다.
- [RUM Explorer][3]의 세션입니다.
- [Error Tracking][4]을 사용하여 다양한 단계, 테스트 및 테스트 실행 전반에 걸쳐 오류를 집계합니다. 오류 영향의 범위를 지정하고 가장 심각한 문제를 해결하는 우선 순위를 지정할 수 있습니다.
- 추가 검색 및 모니터링 기능을 포함한 RUM의 추가 리소스 및 타이밍 정보입니다.

## RUM 애플리케이션에서 Synthetic 데이터 허용

<div class="alert alert-danger">
대상 애플리케이션이 이미 RUM으로 계측된 경우 신서틱(Synthetic) 테스트 설정내에서 RUM 데이터 수집을 활성화하면 안 됩니다. 이렇게 하면 예기치 않은 동작이 발생할 수 있습니다.</div>

브라우저 테스트 기록에서 **Start Recording** 버튼 위의 **Collect RUM Data on**을 클릭하고 데이터를 수집할 애플리케이션을 선택합니다. 기록 및 테스트 구성을 저장한 후 RUM은 테스트 데이터를 수집하고 브라우저 테스트 실행에서 세션 기록을 생성합니다.

1.  **Digital Experience** > **Settings** (Synthetic Monitoring & Testing 내) > [**Integration Settings**][5]로 이동합니다.
2. **Synthetic Data RUM Collection**에서 **Enable Synthetic RUM data collection**을 클릭하여 Datadog이 테스트 실행에서 RUM 데이터를 수집할 수 있도록 합니다.
3. **Save RUM Data Collection**을 클릭합니다.
4.  **Synthetic Data RUM Applications** 아래의 드롭다운 메뉴에서 Browser Test Recorder에 제안된 기본 RUM 애플리케이션을 선택합니다.
5. **Save RUM Data Applications**을 클릭합니다.

(선택 사항) `Synthetic Tests Default`를 클릭하고 드롭다운 메뉴에서 다른 RUM 애플리케이션을 선택합니다. 이 애플리케이션은 브라우저 테스트 기록에 표시됩니다.

자세한 내용은 [Synthetic Monitoring Settings][6]을 참조하세요.

## Synthetics에서 RUM Explorer까지

[브라우저 테스트 목록][7]으로 이동하여 샘플 테스트 실행을 클릭합니다.

{{< img src="synthetics/guide/rum_in_synthetics/browser_test_step_side_panel.png" alt="테스트 단계 세부 정보 사이드 패널" style="width:100%;" >}}

단계 세부 정보 사이드 패널에서는 `This Synthetic test generated data in Real User Monitoring`과 함께 세션 ID 및  **Go to the View in RUM** 버튼이 나타납니다.

{{< img src="synthetics/guide/rum_in_synthetics/generated_rum_data.png" alt="Synthetics Explorer" style="width:80%;" >}}

[RUM Explorer][3]에서 이 테스트 단계의 오류, 리소스 및 성능 데이터를 보려면 **Go to the View in RUM**을 클릭합니다. 또는 **Replay Session** 버튼을 클릭하거나 **View all Session in RUM**을 클릭하여 브라우저 테스트에서 캡처된 사용 가능한 세션 재생을 탐색합니다.

## RUM Explorer에서 Synthetics까지

RUM Explorer에서 [세션 목록][8]으로 이동하여 재생이 가능한 세션을 클릭합니다.

{{< img src="synthetics/guide/rum_in_synthetics/sessions_details_panel.png" alt="세션 세부 정보 사이드 패널" style="width:100%;" >}}

세션 패널에는 `This event was generated by a Synthetic test run`과 함께 테스트 ID 및 **View Synthetic Test Result** 버튼이 나타납니다.

{{< img src="synthetics/guide/rum_in_synthetics/synthetic_test_session.png" alt="세션 세부 정보" style="width:80%;" >}}

Synthetics 및 테스트 결과로 다시 이동하려면 **View Synthetic Test Result**를 클릭합니다.
