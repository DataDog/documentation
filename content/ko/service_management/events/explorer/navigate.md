## 개요

Events Explorer에는 코드 배포, 서비스 상태, 구성 변화, 또는 알림 모니터링과 같이 내 인프라스트럭처와 서비스에 있었던 가장 최근 이벤트가 표시됩니다. 기본적으로 Events Explorer는 이벤트 유형을 표시합니다. 이벤트를 패싯이나 검색 쿼리로 필터링할 수 있습니다.

### 이벤트 사이드 패널

Events Explorer에 있는 이벤트 하나를 클릭하면 이벤트 측면 패널이 열립니다. 여기에서 해당 이벤트와 관련된 태그와 메시지를 확인할 수 있습니다. 

### 속성 탭

{{< img src="service_management/events/explorer/event-attribute.png" alt="Event Attribute 탭 예시" style="width:100%;" >}}

측면 패널의 속성 탭에는 이벤트 속성이 JSON 목록으로 표시됩니다. 속성을 클릭해 기존 쿼리에 추가 또는 제외하거나, 속성의 열을 추가할 수 있습니다.

이벤트 속성은 프로세싱 파이프라인을 사용하여 획득 시 정규화하거나 강화할 수 있습니다.

## 시간 범위

시간 프레임 선택기는 익스플로러 또는 애널리틱스에 표시되는 이벤트 시간 범위를 관리하고, 검색 표시줄 아래의 타임라인 범위를 설정합니다.

시간 프레임 드롭다운에서 프리셋 범위를 선택하거나 커스텀 시간 범위를 입력하여 시간 범위를 변경합니다.

모든 검색 파라미터가 URL 내에 포함되어 있으므로 URL을 공유하여 보기를 공유할 수 있습니다.
