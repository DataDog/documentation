## 개요

스캔 규칙 라이브러리는 이메일 주소, 신용카드 번호, API 키, 인증 토큰 등과 같은 일반적인 패턴을 감지하기 위해 미리 정의된 규칙 모음입니다.

{{< /whatsnext >}}

이러한 규칙은 Datadog에서도 확인할 수 있습니다:

1. 민감한 데이터 스캐너][1]로 이동합니다.
1. 페이지 오른쪽 상단의 **검색 규칙 라이브러리**를 클릭합니다.
1. 라이브러리에서 스캔 그룹에 규칙을 추가하려면 다음을 수행하세요.
  a. 추가하려는 규칙을 선택합니다.   
  b. **스캔 그룹에 규칙 추가**를 클릭합니다.   
  c. 민감한 데이터 스캐너 설정][2]의 단계에 따라 설정을 완료합니다.
