## 팀 상세 정보

각 팀에는 팀, 팀원, 관련 리소스를 포함한 상세 정보 페이지가 있습니다. [팀 디렉터리][1]를 클릭해 팀 상세 정보 페이지로 이동할 수 있습니다.

팀 페이지에는 팀과 관련한 다음 리소스가 표시됩니다.
- 대시보드
- 노트북
- 모니터링
- 서비스
- 링크
- 진행 중인 인시던트
- API 엔드포인트

페이지에서 어떤 리소스를 어떤 순서로 표시할지 정할 수 있습니다.

이모지 아바타와 배너를 선택해 팀 외형을 사용자 지정할 수 있습니다. Datadog에서 이모지 아바타는 팀 이름 옆에 표시됩니다.

팀 외형을 사용자 지정하려면 다음 지침을 참고하세요.

### 레이아웃 사용자 지정

팀 페이지 레이아웃을 수정하려면 `user_access_manage` 또는 `teams_manage` 권한이 있어야 합니다. 또는 팀에서 팀원이나 매니저가 팀 상세 정보를 수정할 수 있도록 구성해야 합니다.

1. **페이지 레이아웃**을 클릭하세요. 페이지 레이아웃 대화 상자가 나타납니다.
1. 리소스를 재정렬하려면 핸들 아이콘을 클릭하고 드래그합니다.
1. 리소스를 숨기려면 항목 하나에 마우스를 올리고 눈 아이콘(**Hide content**)을 클릭합니다.

페이지 레이아웃 하단에 숨긴 리소스가 나타납니다. 리소스를 표시하려면 항목 위에 마우스를 올리고 **Unhide content** 아이콘을 클릭합니다.

### 설정 사용자 지정

팀 상세 정보를 수정하려면 `user_access_manage` 또는 `teams_manage` 권한이 있어야 합니다. 또는 팀에서 팀원이나 매니저가 팀 상세 정보를 수정할 수 있도록 구성해야 합니다.

**설정**을 클릭합니다. 설정 대화 상자가 나타납니다.

설정 대화 상자에 있는 메뉴를 사용해 다음 팀 설정을 사용자 지정합니다.
- 팀원
- 링크
- 이름 및 설명
- 아바타 및 배너
- 페이지 레이아웃
- 권한
- 알림

## 팀 멤버십

팀원을 구분하기 위해 팀 매니저 역할을 할당하세요. 팀원 목록 팀 매니저의 이름 옆에 "TEAM MANAGER" 배지가 나타납니다.

팀 설정 아래에 팀 멤버십을 수정할 수 있는 사용자를 지정합니다. 다음 옵션 중에서 선택할 수 있습니다.
- `user_access_manage` 권한이 있는 사용자만
- 팀 매니저
- 팀 매니저와 팀원
- 조직 내 누구나

`user_access_manage` 권한이 있는 사용자는 팀원을 추가 및 제거하거나 팀 상세 정보를 편집할 수 있는 기본 규칙을 설정할 수 있습니다. 팀 디렉터리 페이지에 있는 **Default Settings** 버튼으로 기본 규칙을 설정하세요. 팀 상세 패널에서 개별 팀 정책을 재정의할 수 있습니다.

## SAML 속성 매핑

SAML 속성을 사용해 팀과 팀 구성원 자격을 관리하려면 [Map SAML 속성을 팀에 매핑][2]을 참고하세요.

## 팀 관리 위임

개방형 멤버십 모델의 경우 기본 팀 설정을 **조직 내 누구나**로 설정하면 조직에 있는 누구나 팀원을 추가하거나 제거할 수 있습니다. 적절한 역할에 `teams_manage` 권한을 할당하면 누구나 팀을 만들거나 팀 상세 정보를 편집할 수 있도록 할 수 있습니다.

팀 중심형 멤버십 모델을 만들고 싶을 경우 기본 팀 설정을 **Team Managers** 또는 **Team Managers and Members**로 설정하여 팀 매니저나 팀 매니저 또는 팀원만 팀원을 추가하거나 제거할 수 있도록 하세요. 팀 매니저 전체를 포함한 역할에 `teams_manage` 역할을 할당하세요.

제한적인 멤버십 모델을 만들고 싶을 경우에는 기본 팀 설정을 **Only users with user_access_manage**로 설정해 **user_access_manage** 권한이 있는 사용자만 팀원을 추가하거나 제거할 수 있도록 하세요. 조직 관리자에게만 `teams_manage` 권한을 할당하세요.
