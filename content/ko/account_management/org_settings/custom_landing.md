## 개요

Datadog 조직 랜딩 페이지는 사용자가 Datadog에 로그인하거나 Datadog 루트 페이지로 이동할 때 표시되는 첫 번째 페이지입니다. Datadog은 조직의 기본 랜딩 페이지를 설정합니다. APM을 사용하는 경우 Datadog은 APM 루트를 랜딩 페이지로 설정합니다. APM을 사용하지 않는 경우 대시보드 목록이 기본 랜딩 페이지가 됩니다.

기본 페이지 대신 Datadog을 사용하면 관리자가 대시보드를 조직의 랜딩 페이지로 설정할 수 있습니다. 커스텀 랜딩 페이지는 대규모 또는 소규모 조직이 해당 사용자를 위한 내러티브를 컨트롤하는 데 도움이 됩니다.

사용자가 Datadog에 처음 로그인할 때 표시할 정보로 대시보드를 맞춤 설정할 수 있습니다. [Organization settings][1]를 사용하여 해당 대시보드를 조직의 커스텀 랜딩 페이지로 설정하세요.

## 커스텀 랜딩 페이지 설정

Datadog 관리자 역할 또는 조직 관리(`org_management`) 권한이 있는 사용자만 조직의 커스텀 랜딩 페이지를 설정할 수 있습니다.

커스텀 랜딩 페이지를 설정하려면 다음 단계를 따르세요.

1. [Organization settings][1]로 이동합니다.
2. 왼쪽 탭에서 **Preferences**를 선택합니다.
3. Datadog 홈페이지 섹션에서 **Dashboard** 버튼을 클릭합니다.
4. 드롭다운 목록을 사용하여 대시보드를 선택합니다.
5. **Save** 버튼을 클릭합니다.

## 기본 랜딩 페이지 사용

Datadog 관리자 역할 또는 조직 관리(`org_management`) 권한이 있는 사용자만 조직의 커스텀 랜딩 페이지를 변경할 수 있습니다.

APM 홈의 기본 랜딩 페이지를 복원하려면 아래 단계를 따르세요.

1. [Organization settings][1]로 이동합니다.
2. 왼쪽 탭에서 **Preferences**를 선택합니다.
3. Datadog 홈페이지 섹션에서 **Default: APM Home** 버튼을 클릭합니다.
4. **Save** 버튼을 클릭합니다.

[1]: https://app.datadoghq.com/organization-settings/
