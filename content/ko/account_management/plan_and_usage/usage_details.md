## 개요

관리자는 왼쪽 하단의 사용자 이름 위를 마우스로 가리켜 [사용량][1] 페이지에 액세스한 뒤 다음으로 이동할 수 있습니다.
`Plan & Usage`--> `Usage`.

사용량 페이지는 제품 카테고리별로 그룹화된 사용량을 표시합니다. 제품 탭으로 이동하여 제품 카테고리별 사용량을 보거나 "전체" 탭으로 이동해 모든 제품에 대한 사용량을 볼 수 있습니다. 각 탭은 다음 정보를 제공합니다.

* 이번 달부터 현재까지 요약
* 총 사용량(현재 및 과거)

특정 제품 탭은 또한 추가 도구를 포함할 수 있습니다.

* 커스텀 메트릭 탭: 상위 커스텀 메트릭
* 로그 관리 탭: 인덱스별 로그 사용량

## 이번 달부터 현재까지 요약

이 섹션은 이번 달부터 현재 시점까지의 사용량을 요약해 보여줍니다. "전체" 탭에서 인프라 호스트, 컨테이너, 커스텀 메트릭, APM 호스트, 로그 및 해당 월에 사용한 기타 플랫폼 구성 요소에 대한 이번 달부터 현재까지의 사용량을 확인할 수 있습니다.

{{< img src="account_management/billing/usage-details-v2-01.png" alt="사용량 요약 - 전체 탭" >}}

제품별 탭에서 제품 카테고리의 제품에 대한 이번 달부터 현재까지의 사용량을 볼 수 있습니다.

{{< img src="account_management/billing/usage-details-v2-02.png" alt="사용량 요약 - 네트워크" >}}

위에 표시된 이번 달부터 현재까지의 사용량은 제품 평가판 등 미청구 사용량을 포함합니다. 대부분의 계정에서 "청구 대상" 사용량을 확인할 수 있습니다. 해당 사용량은 최종 청구서에 포함되는 내역을 표시합니다. "청구 대상" 보기는 약정과 할당에 따른 사용량 외 온디맨드 사용량을 상세하게 보여줍니다. 

{{< img src="account_management/billing/usage-details-v2-07.png" alt="사용량 요약 - 청구 대상" >}}
API 사용자의 경우, 엔드포인트에서 ["전체"][2] 사용량 및 ["청구 대상"][3] 사용량에 액세스할 수 있습니다.

각 제품의 이번 달부터 현재까지 사용량은 아래와 같이 계산됩니다.

| 제품                   | 설명                                                                                                                |
|--------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| 인프라 호스트             | 현재 달의 모든 시간에 대해 모든 고유 인프라 호스트의 99번째 백분위수를 표시합니다.                         |
| 컨테이너               | 현재 달의 모든 시간에 대해 모든 고유 컨테이너의 높은 값(워터마크)을 표시합니다.                                    |
| APM 호스트                | 현재 달의 모든 시간에 대해 모든 고유 APM 호스트의 99번째 백분위수를 표시합니다.                                    |
| 프로파일링된 호스트           | 현재 달의 모든 시간에 대해 프로파일링된 모든 고유 호스트의 99번째 백분위수를 표시합니다.                               |
| 프로파일링된 컨테이너      | 현재 달의 모든 시간에 대해 프로파일링된 모든 고유 컨테이너의 평균을 표시합니다.                                  |
| 커스텀 메트릭           | 현재 달의 모든 시간에 대해 고유 [커스텀 메트릭][4]의 평균 수를 표시합니다.                               |
| 수집된 커스텀 메트릭  | 현재 달의 모든 시간에 대해 수집된 고유 커스텀 메트릭의 평균 수를 표시합니다.                           |
| 수집 로그            | 현재 달의 모든 시간에 대해 수집된 모든 로그 바이트 합계를 표시합니다.                                                |
| 인덱싱된 로그             | 현재 달의 모든 시간에 대해 인덱싱된 모든 로그 이벤트 합계를 표시합니다.                                                |
| 검사된 로그             | 현재 달의 모든 시간에 대해 민감한 데이터 검사기가 검사한 모든 로그 바이트 합계를 보여줍니다.                       |
| 수집된 스팬(span)           | 현재 달의 모든 시간에 대해 수집된 모든 스팬 합계를 표시합니다.                                                    |
| 인덱싱된 스팬            | 현재 달의 모든 시간에 대해 인덱싱된 모든 스팬의 합계를 보여줍니다.                                             |
| 분석된 로그(보안) | 현재 달의 모든 시간에 대해 수집된 모든 분석 로그 바이트 합계를 표시합니다.                                       |
| 서버리스 함수     | 현재 달에서 매 시간 1회 이상 실행된 함수 수의 평균을 보여줍니다.              |
| 서버리스 호출   | 현재 달의 모든 시간에 대한 전체 호출 합계를 보여줍니다.                                                       |
| Fargate 작업            | 현재 달의 모든 시간에 대해 전체 Fargate 작업의 합계를 보여줍니다.                                                     |
| 네트워크 호스트            | 현재 달의 모든 시간에 대해 모든 고유 네트워크 호스트의 99번째 백분위수를 표시합니다.                                |
| 네트워크 플로우            | 현재 달의 모든 시간에 대해 인덱싱된 모든 네트워크 플로우의 합계를 보여줍니다.                                             |
| 네트워크 장치          | 현재 달의 모든 시간에 대해 모든 고유 네트워크 장치의 99번째 백분위수를 보여줍니다.                              |
| 신서틱(Synthetic) API 테스트      | 현재 달의 모든 시간에 대해 모든 신서틱 API 테스트의 합계를 보여줍니다.                                               |
| 신서틱 브라우저 테스트  | 현재 달의 모든 시간에 대한 모든 신서틱 브라우저 테스트 합계를 보여줍니다.                                           |
| RUM 세션             | 현재 달의 모든 시간에 대해 모든 고유 RUM 세션 합계를 표시합니다.                                             |
| 인시던트 관리      | 선택한 달에서 현재 시점까지 인시던트 수명 주기와 타임라인을 이용한 고유 활성 사용자의 수를 보여줍니다.     |
| IoT 장치              | 현재 달의 모든 시간 동안 모든 고유 IoT 장치의 99번째 백분위수를 표시합니다.                                  |


## 사용량 트렌드

[사용량 트렌드][5] 섹션에는 계정 내 모든 조직에 대한 합계 사용량을 표시하는 제품 사용량 그래프가 포함되어 있습니다. 사용 보고서는 **CSV로 다운로드** 버튼을 통해 다운로드할 수 있습니다. 이 보고서에는 각 조직의 제품별 사용량에 대한 시간별 분석이 포함되어 있습니다.

{{< img src="account_management/billing/UsageTrendsOverviewAndCSV.png" alt="CSV로 다운로드 옵션이 강조 표시된 Datadog 애플리케이션의 사용량 트렌드 그래프 페이지" style="width:100%; align:left" >}}

하위 유형을 포함하는 제품의 경우, 해당 제품 그래프에 각 카테고리가 구분되어 있습니다.

{{< img src="account_management/billing/UsageGraphsByProductTab.png" alt="선택된 인프라 탭을 비롯해 인프라 호스트, 에이전트 호스트, 컨테이너 등 인프라 사용량 하위 유형에 대한 여러 그래프를 포함하는 사용량 요약" style="width:100%; align:left" >}}

더 자세한 제품 하위 유형 그래프는 각 제품 탭에서 확인할 수 있습니다. 예를 들어 호스트 유형별 분석은 인프라 탭에서 확인할 수 있습니다.

{{< img src="account_management/billing/UsageBreakdownByProductSubtype.png" alt="에이전트 호스트 및 AWS 호스트가 포함된 인프라 호스트 그래프, 인덱싱된 일일 라이브 로그 및 인덱싱된 누적 라이브 로그가 포함된 인덱싱된 로그 그래프가 있는 인프라 탭의 사용량 트렌드 섹션" style="width:100%; align:left" >}}

합계 기반 제품의 경우 시간에 따른 누적 사용량을 사용할 수 있습니다.

{{< img src="account_management/billing/CumulativeUsageLine.png" alt="수집된 스팬 및 인덱싱된 스팬에 대한 그래프(각각 해당 스팬의 일일 및 누적 합계에 대한 데이터 표시)" style="width:100%; align:left" >}}

시간 선택은 일일, 주간, 월간 및 연간 간격으로 사용량 그래프를 볼 수 있는 옵션을 포함합니다.

{{< img src="account_management/billing/TimeGranularity.png" alt="사용량 그래프의 시간 간격" style="width:100%; align:left" >}}

대시 기호로 연결된 `Committed` 줄은 허용량(예: 커스텀 메트릭 또는 컨테이너) 없이 제품별 약정을 표시합니다.

{{< img src="account_management/billing/CommittedLine.png" alt="값 10으로 구성된 APM 호스트 그래프의 약정 사용량 줄" style="width:100%; align:left" >}}



## 상위 커스텀 메트릭

커스텀 메트릭 탭의 상위 커스텀 메트릭 표는 월별 사용량과 가장 최근 사용량(예: 마지막 업데이트 날짜의 사용량)에 대한 두 가지 보기를 제공합니다.

"상위 5000" 보기는 상위 5000개 커스텀 메트릭에 대한 다음 정보를 제공합니다.
* 메트릭 이름
* 시간당 평균 커스텀 메트릭
* 시간당 최대 커스텀 메트릭
* 전체 커스텀 메트릭 사용량에 대한 메트릭 기여 백분율
* 상위 5000개 커스텀 메트릭 중 메트릭 검색
* 이 데이터는 CSV 파일로 다운로드할 수 있습니다.

"전체" 보기는 모든 커스텀 메트릭에 대해 다음 정보를 제공합니다.
* 메트릭 이름
* 평균 시간당 커스텀 메트릭
* 시간당 최대 커스텀 메트릭
* 모든 커스텀 메트릭 중 메트릭 검색
* 이 데이터는 최대 300,000개의 커스텀 메트릭이 포함된 CSV 파일로 다운로드할 수 있습니다. [API 엔드포인트][6]를 사용하여 300,000개 이상의 커스텀 메트릭을 다운로드할 수 있습니다.


메트릭에 대한 자세한 내용을 보려면 관심 있는 메트릭 행 위로 마우스를 가져간 다음 오른쪽에 표시되는 측정기 아이콘을 클릭하여 [메트릭 요약][7]으로 이동하세요.

## 인덱스별 로그 사용량

로그 관리 탭에서 이 표는 인덱스 이름 및 보존 기간별로 시간별, 일별, 월별 및 연간 인덱스 로그 사용량을 표시합니다. 또한 라이브 로그와 [복원된 로그][8]의 상세 내용을 보여줍니다. 다음 정보가 제공됩니다. 

* 인덱스 이름
* 보존 기간(일)
* 인덱싱된 로그 개수
* 선택한 기간 동안 전체 인덱스 로그 사용량에 대한 인덱스 기여 백분율

이 데이터는 CSV 파일로 다운로드할 수 있습니다.
