## 개요

쿼리는 용어와 연산자로 구성됩니다.

용어에는 다음과 같은 두 가지 유형이 있습니다.

- **단일 용어**는 `test` 또는 `hello`과 같은 단일 단어입니다.
- **시퀀스**는 `"hello dolly"`와 같이 큰따옴표로 묶인 단어의 그룹입니다.

여러 용어를 하나의 복잡한 쿼리로 결합하려면 다음과 같은 불리언 연산자를 사용합니다:

| 연산자 | 설명                                                                                        |
|--------------|------------------------------------------------------------------------------------------------------- |
| `AND`        | **교집합**: 두 용어가 모두 선택한 보기에 포함됩니다. 연산자를 사용하지 않으면 기본값으로 `AND`가 사용됩니다. |
| `OR`         | **합집합**: 두 용어 중 하나가 선택한 보기에 포함됩니다.                                             |
| `-`          | **여집합**: 해당 용어가 선택한 보기에 포함되지 않습니다.                                                  |

## 자동 완성

검색창의 자동 완성 기능을 활용하여 기존 값으로 쿼리를 완성할 수 있습니다.

## 숫자값

특정 범위 내에서 숫자값을 검색할 수 있습니다. 예를 들어 **처리 시간(Duration)** 패싯에서 평균 처리 시간이 2 ~ 10 나노초인 모든 배치를 검색할 수 있습니다. 검색 쿼리는 `Duration:[2-10]`로 업데이트됩니다.
