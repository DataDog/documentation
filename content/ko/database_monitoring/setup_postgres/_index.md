### 에이전트 통합 오버헤드

에이전트 통합 오버헤드 테스트는 Amazon EC2 머신 `c5.xlarge` 인스턴스(4 vCPU, 8GB RAM)에서 진행되었습니다. 테스트에 사용된 데이터베이스는 Amazon RDS `db.m5.large` 인스턴스(2 vCPU, 8GB RAM)에서 동작하는 PostgreSQL 14.10 인스턴스입니다. 해당 데이터베이스는 웨어하우스 20개가 있는 TPC-C 워크로드를 실행하였습니다.

| 설정                           | 수집 간격 |
| --------------------------------- | ------------------- |
| 최소 수집 간격 점검하기     | 15s                 |
| 쿼리 메트릭 수집 간격 | 10s                 |
| 쿼리 샘플 수집 간격 | 10s                 |
| 수집 간격 설정      | 600s                |
| 스키마 수집 간격        | 600s                |

* 에이전트 테스트 버전: `7.50.2`
* CPU: 평균 CPU 사용량의 ~ 0.98%
* 메모리: RAM 사용량의 ~ 290MiB(RSS 메모리)
* 네트워크 대역폭: ~28 KB/s ▼ | 23 KB/s ▲
* 데이터베이스의 에이전트 쿼리 오버헤드:  ~1% CPU 시간
