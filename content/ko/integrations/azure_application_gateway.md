## 개요

Azure Application Gateway는 내 애플리케이션의 트래픽을 관리하도록 해주는 웹 트래픽 로드 밸런서입니다.

Datadog Azure 통합을 사용해 Azure Application Gateway 메트릭을 수집할 수 있습니다.

## 설정

### 설치

아직 설정하지 않았다면, 먼저 [Microsoft Azure 통합][1]을 설정하세요. 그 외 다른 설치 단계는 없습니다.
