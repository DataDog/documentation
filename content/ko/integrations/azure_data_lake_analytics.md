## 개요

Azure Data Lake Analytics는 빅 데이터를 간소화해주는 온디맨드 분석 작업 서비스입니다.

Datadog Azure 통합을 사용해 Data Lake Analytics 메트릭을 수집할 수 있습니다.

## 설정

### 설치

아직 설정하지 않았다면, 먼저 [Microsoft Azure 통합][1]을 설정하세요. 그 외 다른 설치 단계는 없습니다.
