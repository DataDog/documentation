## 개요

Azure Event Grid는 게시-구독 모델을 사용하여 균일한 이벤트 소비를 가능케 하는 완전 관리 지능형 이벤트 라우팅 서비스입니다.

Datadog Azure 통합을 사용해 Azure Event Grid 메트릭을 수집할 수 있습니다.

## 설정

### 설치

아직 설정하지 않았다면, 먼저 [Microsoft Azure 통합][1]을 설정하세요. 그 외 다른 설치 단계는 없습니다.
