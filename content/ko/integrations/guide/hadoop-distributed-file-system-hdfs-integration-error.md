title: Hadoop 분산 파일 시스템(HDFS) 통합 오류
---

HDFS와 통합할 때 오류가 발생하면 먼저 다음 단계에 따라 pip 및 Snakebite를 업그레이드해 보세요.

1. Run: /opt/datadog-agent/embedded/bin/pip install --upgrade pip
2. Run: /opt/datadog-agent/embedded/bin/pip install --upgrade snakebite
3. [에이전트를 재시작하세요][1].

이러한 단계를 실행해도 오류가 해결되지 않으면 [Datadog 지원팀][2]으로 문의하세요.

[2]: /ko/help/
