title: 리소스
---
1. 애플리케이션 성능 모니터링(APM)에서 리소스는 애플리케이션의 특정 도메인(일반적으로 계측된 웹 엔드포인트, 데이터베이스 쿼리 또는 백그라운드 작업)을 뜻합니다.
2. RUM에서 리소스는 일종의 이벤트 유형입니다. 리소스 이벤트는 페이지에 로드된 이미지, XHR, Fetch, CSS 또는 JS 라이브러리에 관하여 생성됩니다.
3. 클라우드 보안 관리 설정 오류에서 리소스는 하나 이상의 제어 규정 준수 여부를 지속적으로 스캔해야 하는 설정 가능한 엔티티입니다. AWS 인스턴스 리소스의 예로는 호스트, 컨테이너, 보안 그룹, 사용자 및 고객 관리 IAM 정책이 있습니다.
