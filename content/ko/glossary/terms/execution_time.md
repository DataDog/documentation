title: 실행 시간
---
APM에서 실행 시간은 스팬이 활성 상태로 간주되거나 하위 스팬이 완료될 때까지 기다리지 않는 총 시간입니다.

실행 시간은 스팬이 활성화된 시간을 합산하여 계산됩니다. 즉, 하위 스팬이 없음을 의미합니다. 비동시 작업의 경우 간단합니다. 다음 이미지에서 스팬 1의 실행 시간은 $\D1 + \D2 + \D3$입니다. 스팬 2와 3의 실행 시간은 각각의 너비입니다.

{{< img src="tracing/visualization/execution-time1.png" style="width:50%;" alt="실행 시간" >}}

하위 스팬이 동시적인 경우, 겹치는 시간을 동시에 활성화된 스팬 수로 나누어 실행 시간을 계산합니다. 다음 이미지에서 스팬 2와 3은 동시적이며(둘 다 스팬 1의 하위 항목임) 스팬 3의 기간 동안 겹치므로 스팬 2의 실행 시간은 $\D2 ÷ 2 + \D3$이고 실행 시간은 스팬 3의 $\D2 ÷ 2$입니다.
