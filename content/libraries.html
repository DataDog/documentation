--- title: Libraries sidebar: nav: - header: Official Libraries - text: Python href: #python - text: Ruby href: #ruby - text: PHP href: #php - header: Community Libraries - text: Java href: #community-java - text: Node.js href: #community-node - text: Perl href: #community-perl - text: Ruby href: #community-ruby - text: PHP href: #community-php --- There are many libraries available to help you interact with the Datadog API.

Python

  • DogApi - A Python Datadog API wrapper.
  • dogstatsd-python - A Python Datadog Statsd client.

Ruby

  • DogApi - A Ruby Datadog API wrapper.
  • dogstatsd-ruby - A Ruby Datadog Statsd client.

PHP

  • php-datadogstatsd - A PHP Statsd client.

Community Libraries

Some great folks have written their own libraries to help interact with Datadog. Check them out:
  • Java
    • java-dogstatsd-client - a Dogstatsd Client for Java written by Indeed.
    • metrics-datadog - a backend to codahale's metrics library written by Vistar Media.
  • Node.js
    • node-datadog - a Node.js API client, contributed by HashGo.
    • node-dogstatsd - a Node.js client DogStatsd client, contributed by Young Han Lee.
    • node-dogapi - a Node.js API client, contributed by Brett Langdon.
  • Perl
    • webservice-datadog - a Perl API client, contributed by Jennifer Pinkham.
    • dogstatsd-perl - a Perl DogStatsd client, contributed by Stefan Goethals.
  • Ruby
    • metricks-dogstatsd - a backend for the popular Metriks gem, written by Mavenlink.
  • PHP
    • plesk_metrics_datadog - a PHP script to collect metrics from Plesk by Israel Viana.

If you've written a Datadog library, write us at code@datadoghq.com and we'll be happy to add it to the list.
