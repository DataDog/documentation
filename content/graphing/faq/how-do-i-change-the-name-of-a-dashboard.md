## Timeboard

If you want to change the name of a Timeboard:

* Click on the info icon on the top right corner of the dashboard
* Click on the pencil icon to edit the title and description
* Click the checkmark to save changes

## Screenboard

If you want to change the name of a Screenboard:

* Click on Edit Board at the top of the dashboard
* Change the name to whatever you like
* Click Save Changes
