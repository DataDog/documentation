## Add Tags To A Host
This end point allows you to add tags to a host.

##### ARGUMENTS

* `tags` [*required*]:  
    A list of tags to apply to the host
* `source` [*optional*, *default*=**users**]:  
    The source of the tags (e.g. chef, puppet).
