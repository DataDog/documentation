## Update host tags
This end point allows you to update all tags for a given host.

##### ARGUMENTS
* **`tags`** [*required*]:  
    A list of tags
* **`source`** [*optional*, *default*=**users**]:  
    The source of the tags (e.g. chef, puppet).  
    [Complete list of source attribute values][1]
