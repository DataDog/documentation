## Supprimer les tags d'un hôte
Cet endpoint vous permet de supprimer tous les tags d'un hôte donné.

##### ARGUMENTS
* **`source`** [*facultatif*, *défaut* = **users**] :
    La source des tags (p. ex., chef, puppet).
