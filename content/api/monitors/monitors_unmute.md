## Unmute A Monitor

##### ARGUMENTS
* `scope` [*optional*]:  
    The scope to apply the mute to.  
    For example, if your alert is grouped by {host}, you might mute 'host:app1'
* `all_scopes` [*optional*]:  
    Default = `False`
    Clear muting across all scopes
