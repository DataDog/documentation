## Mute a monitor

##### ARGUMENTS
* `scope` [*optional*, *default*=**None**]:  
    The scope to apply the mute to, e.g. **role:db**.
* `end` [*optional*, *default*=**None**]:  
    A POSIX timestamp for when the mute should end.
