## Désactiver un monitor

##### ARGUMENTS
* **`scope`** [*optionnel*, *défaut*=**None**]:  
    Le context auquel appliquer le silence, ex. **role:db**.
* **`end`** [*optionnel*, *défaut*=**None**]:  
    Timestamp POSIX de la fin du mute.
