Le dashboard d'erreurs fournit des informations sur les erreurs de vos applications. Il est divisé en deux sections :

- **General errors** :
    Visualisez les pages, les navigateurs et les pays qui sont les plus affectés par les erreurs frontend.
- **Resource errors** :
    Surveillez vos ressources XHR et Fetch avec un code de statut supérieur ou égal à `400`.

Pour en savoir plus sur les informations affichées, consultez la [documentation Données RUM collectées] [1]
