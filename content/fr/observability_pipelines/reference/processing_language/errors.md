Datadog Processing Language (DPL), ou Vector Remap Language (VRL), est un langage « fail-safe », ce qui signifie qu'un programme DPL/VRL se compile uniquement si toutes les erreurs potentielles ont été gérées. Vos programmes DPL/VRL peuvent ainsi régler les problèmes de mise en forme de données.
