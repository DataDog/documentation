title: pipeline de processing
---
Dans le cadre des événements Datadog, un pipeline de processing est une séquence définie d'actions de structuration de données exécutées sur les attributs des événements au moment de leur ingestion. Les utilisateurs peuvent configurer des pipelines de processing afin de normaliser et d'enrichir les événements.
Pour en savoir plus, <a href="https://docs.datadoghq.com/events/#normalize-and-enrich-events-with-processing-pipelines">consultez la documentation</a>.
