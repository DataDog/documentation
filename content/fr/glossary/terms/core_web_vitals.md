title: Signaux Web essentiels
---
Les signaux Web essentiels sont un ensemble de métriques que Google identifie comme des signaux utiles pour les tests de lʼUX. Ces métriques concernent le temps de chargement d'une page, le délai d'interaction de l'utilisateur, la stabilité d'une page pendant son chargement, etc. Pour en savoir plus, <a href="https://support.google.com/webmasters/answer/9205520?hl=en/">consultez la documentation à ce sujet</a>.
