title: Network Device Monitoring (NDM)
---
Network Device Monitoring (NDM) de Datadog offre une visibilité sur les périphériques réseau sur site et virtuels, tels que les routeurs, les commutateurs et les pare-feu.
Pour obtenir plus d'informations, <a href="/network_monitoring/devices">consultez la documentation à ce sujet</a>.
