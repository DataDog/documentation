title: collection de tests

Dans le service de surveillance Datadog Synthetic et dans CI Visibility, une collection de tests désigne généralement un groupe de tests portant sur la même unité de code selon votre langage et votre framework de test. Vous pouvez consulter un <a href="https://github.com/DataDog/datadog-ci/blob/master/src/commands/junit/__tests__/upload.test.ts">exemple de collection de tests</a>, disponible sous forme de fichier test situé dans le référentiel `datadog-ci`. Pour en savoir plus, <a href="/tests/#fonctionnalites-prises-en-charge">consultez la documentation</a>.
