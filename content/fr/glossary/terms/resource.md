title: ressource
---
1. Dans lʼAPM, une ressource est un domaine particulier d'une application. Il s'agit généralement d'un endpoint web instrumenté, d'une requête de base de données ou d'une tâche en arrière-plan.
2. Dans la solution RUM, une ressource est un type dʼévénement. Un événement de ressource est généré pour les les bibliothèques dʼimages, XHR, Fetch, CSS ou JS chargées sur une page.
3. Dans Cloud Security Management Misconfigurations, une ressource est une entité configurable devant être analysée en continu afin de confirmer qu'elle passe un ou plusieurs contrôles. Exemples de ressource d'instance AWS : hosts, conteneurs, groupes de sécurité, utilisateurs et stratégies IAM gérées par le client.
