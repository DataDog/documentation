title: code minifié
---
Le code minifié est un code (souvent JavaScript) dépourvu de commentaires, d'espaces blancs supplémentaires, de code inutilisé et de tout ce qui n'affecte pas la fonctionnalité. Le code minimisé est moins lisible par l'homme, mais sa taille réduite améliore les performances des sites web.
