## Intégration GCP

<mark>Les endpoints GCP ne sont pas pris en charge dans les bibliothèques client de Datadog. Pour demander cette fonctionnalité, contactez l’[assistance Datadog][1].</mark>

Configurez votre intégration Datadog/Google Cloud Platform (GCP) directement avec l'API Datadog.  
[En savoir plus sur l'intégration Datadog/Google Cloud Platform][2].
