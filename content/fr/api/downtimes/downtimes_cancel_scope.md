## Annuler les downtimes d'un monitor selon le contexte

<mark>Ce endpoint n'est pas pris en charge dans les bibliothèques client de Datadog. Pour demander cette fonctionnalité, contactez l’[assistance Datadog][1].</mark>

**ARGUMENTS**:

* **`scope`** [*obligatoire*] :
    annulez tous les downtimes en fonction d'un ou plusieurs contextes donnés, comme :

    *  **env:prod**
    *  **role:db,role:db-slave**
