## Obtenir la séquence des index

<div class="alert alert-warning">
Cet endpoint est en bêta public. Si vous souhaitez nous faire part de vos remarques, <a href="/help">contactez l'assistance Datadog</a>.
</div>

Utilisez les endpoints de séquence des index pour contrôler la _séquence_ de processing des index d'une organisation. Utilisez cet endpoint uniquement si vous avez plus d'un index de logs dans votre organisation. Notez que cet index ne peut pas être utilisé pour ajouter ou supprimer des index.

**ARGUMENTS**:

Cet endpoint ne prend aucun argument JSON.
