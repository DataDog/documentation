## Intégration AWS

<mark>Les endpoints AWS ne sont pas pris en charge dans les bibliothèques client de Datadog. Pour demander cette fonctionnalité, contactez l’[assistance Datadog][1].</mark>

Configurez votre intégration Datadog/AWS directement avec l'API Datadog.  
Pour en savoir plus, consultez la [page sur l'intégration AWS][2].
