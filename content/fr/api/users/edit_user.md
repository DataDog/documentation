## Modifier un utilisateur

_Ne peut être utilisé qu'avec des clés d'application confiées aux administrateurs._

Modifie les attributs d'un utilisateur.

**ARGUMENTS** :

* **`user_id`** [*obligatoire*] : ID de l'utilisateur à modifier.
* **`disabled`** [*facultatif*] : indique si l'utilisateur est activé ou désactivé.
* **`email`** [*facultatif*] : l'adresse e-mail associée à l'utilisateur.
* **`name`** [*facultatif*] : le nom associé à l'utilisateur.
