Le widget Sankey affiche le flux, les déplacements ou les transitions des utilisateurs d'un état d'application à un autre. Ce widget est disponible pour Product Analytics.

## Configuration

### Configuration

1. Définissez le diagramme Sankey et sélectionnez parmi les options du menu déroulant pour afficher les étapes suivies par les utilisateurs avant ou après avoir visité une vue donnée.
1. Filtrez vos données par segment d'utilisateur ou d'autres attributs de tags d'application.
1. Définissez les options d'affichage pour afficher `N` vues par étape et personnalisez votre graphique pour trier par nombre de sessions et afficher les chemins vers d'autres vues.
