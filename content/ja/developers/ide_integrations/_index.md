## 概要

Datadog インテグレーションをお好みの統合開発環境 (IDE) で使用し、コーディングしながら Datadog サービスと対話することができます。

    {{< nextlink href="developers/ide_integrations/vscode/" >}}<u>Visual Studio Code</u>: VS Code の Synthetic テストをローカル環境で実行することにより、コードの信頼性を向上させます。{{< /nextlink >}}
