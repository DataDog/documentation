## 概要

Datadog インテグレーションをお好みの統合開発環境 (IDE) で使用し、コーディングしながら Datadog サービスと対話することができます。

    {{< nextlink href="developers/ide_plugins/idea/" >}}<u>JetBrains IDEs</u>: リアルタイムの観測可能性データに基づいて IDEA および GoLand でコードレベルの有意義な洞察を提供することで、ソフトウェアのパフォーマンスを向上させます。{{< /nextlink >}}
    {{< nextlink href="developers/ide_plugins/vscode/" >}}<u>Visual Studio Code</u>: この拡張機能は、Code Insights とローカル環境で Synthetic テストを実行するためのインテグレーションを備えています。{{< /nextlink >}}
    {{< nextlink href="developers/ide_plugins/visual_studio/" >}}<u>Visual Studio</u>: .NET 開発者のための Datadog 拡張機能。{{< /nextlink >}}
