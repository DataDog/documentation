## 概要

Azure Table Storage は、大量の半構造化データセットを使用して迅速な開発を行うための NoSQL key-value ストアです。

Azure Table Storage からメトリクスを取得すると、以下のことができます。

- テーブルストレージのパフォーマンスを視覚化できます。
- テーブルストレージのパフォーマンスをアプリケーションと関連付けることができます。
