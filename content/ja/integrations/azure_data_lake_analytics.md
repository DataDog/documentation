## 概要

Azure Data Lake Analytics は、ビッグデータを簡略化するオンデマンド分析ジョブサービスです。

Datadog Azure インテグレーションを使用して、Data Lake Analytics からメトリクスを収集できます。

## セットアップ

### インストール

[Microsoft Azure インテグレーション][1]をまだセットアップしていない場合は、最初にセットアップします。それ以上のインストール手順はありません。
