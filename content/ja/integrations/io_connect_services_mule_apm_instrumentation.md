## 概要

Datadog Mule® Integration for APM を使用して、当社のエンジニアがお客様の環境内で Mule アプリケーションにインスツルメントを施し、トレースを作成して Datadog APM に送信することで、24 時間 365 日のエンドツーエンドの可視性を提供します。

Datadog のゴールドパートナーであり、Datadog Mule® Integration および Datadog Mule® Integration for APM の開発元である IO Connect Services は、エンタープライズグレードのソリューション構築と実装に関して豊富で深い専門知識を有しています。

当社の専門家は次のことを行います。
 * お客様の Mule アプリケーションおよび API におけるトレース戦略を策定します。
 * 最大 3 つの環境 (開発、QA、本番) に [Datadog Mule® Integration for APM][1] を
インストールします。
 * APM を使用して Mule アプリケーションをインスツルメントし、Datadog にトレースを作成します。
 * レポートやダッシュボードの作成に役立つよう、トレースやスパンにメタデータを追加します。
 * スパンのステータスや HTTP コードを適切にマークします。
 * ログを APM トレースに相関付けます。
