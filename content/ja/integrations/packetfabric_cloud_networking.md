## 概要

PacketFabric は、グローバルなネットワーク・アズ・ア・サービスを提供しています。主要なキャリアニュートラルなコロケーション施設と AWS、Microsoft Azure、Google Cloud などの大手クラウドサービスプロバイダー間で、動的かつリアルタイムのネットワーク接続を実現します。PacketFabric は、稼働率 99.999% が保証されたサービスレベル契約のもと、プライベートかつセキュアなキャリアグレードの 70TB 以上のネットワークを運用しています。お客様は、セルフサービスプラットフォームを通じて、ホスティングまたは専用クラウド接続や Cloud Router (CR) などのネットワークサービスを注文し、数か月ではなく数分で利用を開始することができます。

企業は PacketFabric を活用して WAN バックボーン全体を運用したり、ハイブリッドクラウドやマルチクラウドのインフラストラクチャーを接続したりすることで、アプリケーション開発を推進し、アプリケーションのパフォーマンスを向上させることができます。

Datadog を使用する PacketFabric のお客様は、Datadog 内でリアルタイムのネットワークメトリクスを確認することができます。

詳細については、[PacketFabric][1] をご覧ください。
