## 説明
基本的なタイプ (文字列、整数、浮動小数点数) を比較する場合、常に同じタイプの値を比較する必要があります。

## 非準拠コードの例
```python
1 == "1"  # 整数と文字列の比較
1.0 == "foo"  # 浮動小数点数と文字列の比較
```

## 準拠コードの例
```python
1 == 1
"abc" == "def"
a == 1
a == b
```
