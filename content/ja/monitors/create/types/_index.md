{{< whatsnext desc="モニターの種類を選択:">}}
{{< nextlink href="/monitors/create/types/host" >}}<u>Host</u>: Datadog に報告を行っているホストがあるかどうかをチェック。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/metric" >}}<u>Metric</u>: メトリクスの値をユーザー定義のしきい値と比較。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/anomaly" >}}<u>Anomaly</u>: メトリクスの異常動作を履歴データに基づいて検出。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/outlier" >}}<u>Outlier</u>: グループ内の他のメンバーと挙動が異なるメンバーについてアラート。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/forecasts" >}}<u>Forecast</u>: メトリクスがしきい値を超えて送信されると予測されるとアラート。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/integration" >}}<u>Integration</u>: 特定のインテグレーションのメトリクス値または健全性ステータスを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/process" >}}<u>Live Process</u>: ホストで実行中のプロセスがあるかどうかをチェック。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/process_check" >}}<u>Process Check</u>: process.up サービスチェックで生成されたステータスを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/network" >}}<u>Network</u>: TCP/HTTP エンドポイントのステータスをチェック。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/custom_check" >}}<u>Custom Check</u>: 任意のカスタムチェックのステータスを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/event" >}}<u>Event</u>: Datadog によって収集されたイベントを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/log" >}}<u>Logs</u>: Datadog によって収集されたログを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/apm" >}}<u>APM</u>: APM メトリクスをユーザー定義しきい値と比較。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/real_user_monitoring" >}}<u>Real User Monitoring</u>: Datadog によって収集されたリアルなユーザーデータを監視。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/watchdog" >}}<u>Watchdog</u>: Watchdog が異常動作を検出した場合に通知を受信。{{< /nextlink >}}
{{< nextlink href="/monitors/create/types/composite" >}}<u>Composite</u>: 複数のモニターを組み合わせた式に対してアラート。{{< /nextlink >}}
