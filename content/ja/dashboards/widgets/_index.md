## 概要 {#overview}

ダッシュボードウィジェットはデータの視覚表現です。これらは、インフラストラクチャー全体のデータを視覚化して相関させるための[ダッシュボード][2]の構成要素として機能します。グラフ、画像、ログ、ステータスなど、さまざまな情報タイプを含めることができ、システムや環境の概要を提供します。

## 始める {#get-started}

データに関連するウィジェットを迅速にオンボードする最も簡単な方法は、[プリセットリスト][1]からダッシュボードを複製することです。このリストには、組織の他のメンバーが作成したダッシュボードや、インストールされた統合用のすぐに使えるテンプレートが含まれています。ダッシュボードを複製した後、ウィジェットをユースケースに合わせてカスタマイズできます。

### ウィジェットをダッシュボードに追加する {#add-a-widget-to-your-dashboard}

ダッシュボードでウィジェットを使用するには

1. Datadog の[ダッシュボードリスト][1]に移動します。
2. {{< ui >}}New Dashboard{{< /ui >}} をクリックするか、既存のダッシュボードを選択して編集します。
3. {{< ui >}}Add Widget{{< /ui >}} をクリックします。時系列、棒グラフ、テーブル、イベントストリームなど、さまざまなウィジェットタイプから選択します。
4. ウィジェットを構成します。
    - データソースの選択: メトリクス、ログ、トレース、またはその他のデータソースを選択します。
    - 視覚化のカスタマイズ: 表示設定、単位、時間枠を調整してニーズに合わせます。
    - コンテキストの追加: カスタムリンク、条件付き書式、グループ化を使用してインサイトを強化します。
5. ダッシュボードを保存し、必要に応じてチームや外部と共有します。

詳しくは、[ウィジェット構成][3]を確認し、利用可能な[ウィジェットタイプ][4]をチェックします。

### タブでウィジェットを整理する {#organize-widgets-with-tabs}

ダッシュボードが成長してきたら、タブを使用してウィジェットを名前付きセクションにグループ化します。編集モードでウィジェットの共有メニューを開き、**タブに移動**を選択して既存のタブに割り当てるか、新しいタブを作成します。タブはダッシュボード上部にナビゲーションバーとして表示され、閲覧者が必要なセクションに直接移動できるようになります。詳しくは、[タブ][5]を参照してください。

## データソース {#data-sources}

ウィジェットは、以下を含む複数の Datadog ソースからデータを視覚化できます。

- **APM トレース**: アプリケーションパフォーマンスモニタリングデータ
- **イベント**: カスタムイベント、デプロイメント、およびアノテーション
- **ログ**: ログイベント、ログ分析、およびログベースのメトリクス
- **メトリクス**: インフラストラクチャー、アプリケーション、およびカスタムメトリクス
- **RUM**: Real User Monitoring および Synthetic テストデータ
- **SLO**: Service Level Objectives およびエラー予算
- **セキュリティ**: セキュリティシグナルおよびコンプライアンスデータ

## 一般的なユースケース {#common-use-cases}

{{% collapse-content title="Infrastructure Monitoring" level="h4" expanded=false %}}
- **時系列**ウィジェットを使用して、CPU、メモリ、およびネットワークのメトリクスを時間経過に伴って表示する
- **ホストマップ**ウィジェットを使用して、インフラストラクチャー全体のリソース使用量を視覚化する
- **トップリスト**ウィジェットを使用して、最もリソースを消費するホストやサービスを特定する
{{% /collapse-content %}}

{{% collapse-content title="アプリケーションパフォーマンス" level="h4" expanded=false %}}
- **時系列**ウィジェットを使用して、応答時間、エラー率、スループットを追跡する
- **サービスサマリー**ウィジェットを使用して、サービスの健全性を高レベルで把握する
- **トポロジーマップ**ウィジェットを使用して、サービスの依存関係とデータフローを視覚化する
{{% /collapse-content %}}

{{% collapse-content title="ビジネスインテリジェンス" level="h4" expanded=false %}}
- **クエリ値**ウィジェットを使用して、主要なパフォーマンス指標やビジネスメトリクスのために活用する
- **ファネル**ウィジェットを使用して、アプリケーションを通じてユーザーコンバージョンを追跡する
- **保存期間**ウィジェットを使用して、ユーザーのエンゲージメントと離脱を分析する
{{% /collapse-content %}}

{{% collapse-content title="Incident Response" level="h4" expanded=false %}}
- **アラートグラフ**ウィジェットを使用して、アラートの履歴とトレンドを表示する
- **モニターサマリー**ウィジェットを使用して、インフラストラクチャー全体の現在のアラートステータスを把握する
- **イベントストリーム**ウィジェットを使用して、リアルタイムのイベントモニタリングに活用する
{{% /collapse-content %}}
