## 概要

ウィジェットの構成は、効果的なダッシュボードを作成し、インフラストラクチャーとアプリケーションに関する有意義なインサイトを得るために不可欠です。このガイドでは、ウィジェットを設定する際に理解しておくべき主要な構成オプションと考慮事項について説明します。


## 全画面

全画面モードでは、ウィジェットの表示と分析機能が強化されます。ほとんどのウィジェットが全画面モードでの表示が可能で、以下の操作を行うことができます。

* タイムフレームを変更する
* 選択したタイムフレームだけ前後に移動する
* 現在の時間にグラフを一時停止するか、ライブグラフを表示する
* タイムフレームをリセットする
* ダッシュボードやノートブックにグラフをエクスポートしたり、クエリをコピーする
* グラフを生成するデータを CSV 形式でダウンロードする

ウィジェットの概要に直接アクセスするには、ウィジェットの右上にあるフルスクリーンボタンをクリックします。

[時系列ウィジェット][1]では追加のオプションが利用可能です。

## データのプレビュー

グラフ エディターで複数のデータ ソースを結合して、コンテキストとメタ データを追加することで、より充実した視覚化が可能になります。データのプレビューを使用すると、結合しているデータや、クエリが期待通りに動作しているかどうかを確認できます。この機能は次のような場合に役立ちます。

- データ構造と列名を確認する
- 一致するキーを特定する
- クエリ全体を実行する前に結果を検証する

{{% collapse-content title="Example" level="h4" expanded=false %}}
支払いログを、商品情報を含む参照テーブル (ルックアップ テーブル) と結合して、販売価格や発売日をトランザクション データと一緒に表示することができます。あるいは、Salesforce や Snowflake などの外部ソースから取得した顧客情報と結合して RUM セッション データをリッチ化し、顧客を階層別にセグメント化することもできます。

結合可能なデータ ソースには次のようなものがあります (これらに限定されるわけではありません)。

- ログ
- メトリクス
- RUM
- Cloud Cost Recommendations
- Netflow
- APM スパン
- APM トレース
- プロファイル
- Synthetics CI バッチ
- Synthetics テスト実行
- Static Analysis
- CI テスト
- コンプライアンス診断結果
- プロダクトアナリティクス
- 参照テーブル

結合とデータのプレビューを使用することで、より簡単に適切なフィールドを選択し、関連する情報でグラフを充実させることができ、ダッシュボードのクオリティと有用性が向上します。

## カスタムリンク

カスタム リンクは、ウィジェットのデータ値を Datadog のページや AWS コンソールなどの関連 URL に接続することで、データとのインタラクションを強化します。

一般的なウィジェットのインライン データとのインタラクションをカスタマイズするには、[コンテキスト リンク][2]を参照してください。

## メトリクス情報

メトリクスのグラフ上で、コンテキスト メニュー (縦に並んだ 3 つのドット) をクリックして、**Metrics Info** オプションを見つけます。このオプションを選択すると、メトリクスの説明を含むパネルが開きます。このパネルでメトリクス名をクリックすると、メトリクス サマリー ページでメトリクスが開き、さらなる分析や編集が可能です。

## 単位のオーバーライド

単位のオーバーライドは重要な表示オプションで、ウィジェット上でのデータ値の表示方法をカスタマイズし、データに意味のあるコンテキストを追加することができます。その他の使用例や情報については、[単位オーバーライドで可視化をカスタマイズする][3]を参照してください。
- **Unit override**: 選択することで、「メモリ」ファミリーの単位が表示され、Datadog がデータに応じて適切なスケールを表示するようにします (メガバイトやギガバイトなど)。
- **Unit and scale override**: 単位を単一のスケールに固定します (値に関係なくデータをメガバイトで表示)。
- **Define custom units**: 完全にカスタマイズされた単位を定義します (一般的なカウントの代わりに「テスト」のような)。

これは、データに単位を割り当てるための代替手段ではありません。

## グローバルタイムセレクター

グローバル タイム セレクターは、ダッシュボード上のすべてのウィジェットを同期させて同じタイム フレームを使用できる、基本的な時間設定オプションです。グローバル タイム セレクターを使うには、1 つ以上の時間ベースのウィジェットが `Global Time` を使うように設定されている必要があります。ウィジェット エディターの **Set display preferences** で選択するか、ウィジェットを追加します (`Global Time` はデフォルトの時間設定です)。

グローバル タイム セレクターは、同一のダッシュボード上で `Global Time` オプションを使用するすべてのウィジェットに対して同一のタイム フレームを設定します。過去の動的なタイム ウィンドウ (例: `Past 1 Hour`、`Past 1 Day`) を選択するか、`Select from calendar...` オプションを使用して固定期間を選択します。または、[カスタム タイム フレームを入力][8]します。動的なタイム ウィンドウを選択した場合、ウィジェットはタイム ウィンドウに沿って更新されます。

グローバルタイムにリンクされていないウィジェットは、グローバルウィンドウに適用されたローカルタイムフレームに対応するデータを表示します。たとえば、グローバルタイムセレクターが 2019 年 1 月 1 日から 2019 年 1 月 2 日に設定されている場合、ローカルタイムフレームが `Past 1 Minute` に設定されているウィジェットには、2019 年 1 月 2 日の午後 11 時 59 分からの最後の 1 分が表示されます。

## ウィジェットのコピーと貼り付け

コピー＆ペースト機能は、共有やコラボレーションを可能にする重要な機能で、Datadog のさまざまなコンテキストや外部ツールでウィジェットを再利用できます。

<div class="alert alert-danger">この機能を使用するには、組織設定で<a href="https://app.datadoghq.com/organization-settings/public-sharing/settings"><strong>静的パブリック データ共有</strong></a>を有効にします。</div>

ウィジェットは[ダッシュボード][4]、[ノートブック][5]、[APM サービス][6]、および [APM リソース][7] ページ上で、<kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>C</kbd>を使用するか、共有アイコンから "Copy" を選択することでコピーできます。

コピーされたウィジェットは、<kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>V</kbd>を使用して Datadog 内に貼り付けることができます。

* **ダッシュボード**: マウスカーソルの下に新しいウィジェットを追加します。
* **ノートブック**: ノートブックの最後に新しいセルを追加します。

また、リンクのプレビューを表示するお気に入りのチャットプログラム（Slack や Microsoft Teams など）にウィジェットを貼り付けることもできます。これにより、グラフのスナップショット画像とウィジェットへの直接リンクが表示されます。

詳細は、[Datadog クリップボード][9]を参照してください。

## ウィジェットのグループ

タイムボード グループ ウィジェットをコピーするには、グループ ウィジェット 領域にカーソルを合わせて <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>C</kbd> を使用するか、共有アイコンから "Copy" を選択します。

**注**: スクリーンボードまたはノートブックにグラフを貼り付ける場合、Datadog はグループ内の各ウィジェットを個別に貼り付けます。

複数のスクリーンボード ウィジェットをコピーするには (編集モードのみ)、<kbd>Shift</kbd> を押しながらウィジェットをクリックし、<kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>C</kbd> を使用します。

**注**: これは、Datadog 内で共有する場合にのみ機能します。プレビュー画像は生成されません。

## ウィジェット グラフのメニュー

ウィジェット グラフのメニューでは、データとのインタラクションに不可欠なオプションが提供されています。グラフ上にカーソルを置くと、以下のメニュー オプションにアクセスできます。

### View in full screen

グラフを[全画面モード](#full-screen)で表示します。

### エクスポート

ダッシュボード グラフのエクスポート アイコンをクリックすると、オプション メニューが開きます。

| オプション                 | 説明                                                         |
|------------------------|--------------------------------------------------------------------|
| コピー                   | ダッシュボード グラフのコピーを作成します。                              |
| スナップショットを共有         | グラフのスナップショットを作成および送信します。                          |

#### Datadog 内で使用する場合

| オプション                 | 説明                                                         |
|------------------------|--------------------------------------------------------------------|
| インシデントを宣言       | グラフからインシデントを宣言します。                                |
| インシデントに追加        | 既存のインシデントにグラフを追加します。                             |
| ケースを作成            | グラフからケースを作成します。                                      |
| ノートブックに保存       | グラフをノートブックに保存します。                                      |
| ウィジェットへのリンク         | ウィジェットへのリンクをクリップボードにコピーします。                 |

#### 外部と共有

| 形式 | 説明             |
| -----  | ----------------------- |
| PNG としてダウンロード    | ウィジェットを PNG 形式でダウンロードします。 |
| SVG としてダウンロード    | ウィジェットを SVG 形式でダウンロードします。 |
| CSV としてダウンロード    | ウィジェットを CSV 形式でダウンロードします。 |

### 編集

編集するには、ダッシュボード グラフの鉛筆アイコンをクリックします。

### その他のオプション

ダッシュボード グラフのコンテキスト メニュー (縦に並んだ 3 つのドット) をクリックすると、オプション メニューが開きます。

| オプション                 | 説明                                                         |
|------------------------|--------------------------------------------------------------------|
| 編集                              | グラフを編集します。                                         |
| Clone                             | グラフのコピーを隣に作成します。                   |
| スプリットグラフ                       | [スプリット グラフ][10]を作成します。                             |
| カスタム リンクを作成               | [カスタム リンク](#custom-links)を作成します。                   |
| モニターを作成                    | グラフのクエリーを使用して事前構成されたモニターを作成します。  |
| Metrics info (メトリクス グラフのみ) | このグラフ内のメトリクスの説明を取得します。メトリクス名をクリックすると、Metric Summary ページが開きます。            |
| 削除                            | グラフを削除します。                                       |
