## 概要

Datadog Mobile Real User Monitoring (Mobile RUM) は、Android、iOS、React Native、Flutter、Rokuなどのモバイルアプリケーションのパフォーマンスに関するインサイトを提供します。アプリの安定性を可視化し、アプリのクラッシュを特定して対処し、ユーザーエクスペリエンスに影響を与える可能性のあるあらゆる不具合を評価することができます。

Datadogは、各ユーザージャーニーを、関連する合成テスト、バックエンドメトリクス、トレース、ログ、ネットワークパフォーマンスと関連付けるので、ユーザエクスペリエンスの低下を迅速に検出して解決し、リリースを最適化して、高パフォーマンスのアプリケーションをユーザーに提供できます。

## 開始する
