Datadog では、お客様のニーズに合うさまざまな料金プランを用意しています。詳細については、[料金プラン][1]のページを参照してください。

## 料金の計算

注文書に特に記載のない限り、料金は、カレンダー月ごとの製品使用状況に基づいて計算されます。最も一般的な価格設定単位を以下に示します。

### インフラストラクチャーの監視

* **ホスト**は、1 つの物理または仮想オペレーティングシステムインスタンスです。Datadog は、お客様が Datadog インフラストラクチャーサービスで同時に監視しているホスト数を 1 時間に一度記録します。
   * 最高水準プラン (HWMP) では、月末に、この 1 時間ごとの測定結果が最高から最低まで順序付けられ、そのうちの 8 番目に高い測定値に基づいて課金されます。
  * 月単位/時間単位混合プラン (MHP) では、月単位の最低契約量に対して課金され、ホスト数がその契約量を超えた時間については、時間単位の料金が課金されます。
* **コンテナ**は、アプリケーションソフトウェアと限定的なオペレーティングシステムライブラリおよび設定からなる自己完結型のオペレーティング環境です。Datadog は、お客様が Datadog インフラストラクチャーサービスで同時に監視しているコンテナ数を 5 分間隔で記録します。監視されたコンテナの時間数 (端数を含む) に基づいて月単位で課金されます。
* [**カスタムメトリクス**][2]とは、メトリクス名、ホスト ID、任意のタグからなる単一かつ一意の組み合わせです。1 時間に Datadog インフラストラクチャーサービスに送信されたユニークカスタムメトリクス数の月平均に基づいて課金されます。
* **デバイス**は、1 フレーム内の 1 つ以上のシングルボードコンピューターで構成される物理センサーです。お客様が Datadog インフラストラクチャーサービスで同時に監視しているデバイス数およびホスト数を記録し、それに対して課金されます。
* **クラウド関数**は、定義されたイベントまたはアクションに応答して、クラウドプラットフォームのサーバーレスコンピューティングサービス上で実行するように設定されたアプリケーションコードです (AWS Lambda、Google Cloud Function、Azure Function など)。Datadog インフラストラクチャーサービスは、1 回以上実行または呼び出された関数の数を 1 時間ごとに記録します。月内の全時間の関数の平均数に基づいて課金されます。
* AWS **Fargate タスク**は、AWS の ECS コンテナオーケストレーションプラットフォームを通じてセットアップされたコンテナのコレクションです。Datadog は、お客様が Datadog インフラストラクチャー (または APM) サービスで監視しているタスクインスタンス数を 5 分間隔で記録します。月末にこの間隔に基づく測定値を集計し、アプリケーションが実行されて監視された合計時間数に基づいて課金されます。

### APM

* Datadog は、お客様が Datadog APM サービスで同時に監視している **APM ホスト**数を 1 時間に一度記録します。
   * 最高水準プラン (HWMP) では、月末に、この 1 時間ごとの測定結果が最高から最低まで順序付けられ、そのうちの 8 番目に高い測定値に基づいて課金されます。
  * 月単位/時間単位混合プラン (MHP) では、月単位の最低契約量に対して課金され、ホスト数がその契約量を超えた時間については、時間単位の料金が課金されます。
* **トレースイベント**は、アプリケーションに対する個別のリクエストです。Datadog APM サービスに送信されたトレースイベント数の合計に基づいて課金されます。

### ログ管理

* **ログ**は、オペレーティングシステム、アプリケーション、または他のソースによって生成されたテキストベースのアクティビティ記録です。Datadog ログサービスに送信されたギガバイト数の合計に基づいて、収集されたログに課金されます。
* **ログイベント**は、Datadog ログサービスによってインデックス化された 1 つのログです。お客様が選択した保持ポリシーに指定されている料金で、インデックス化のために送信されたログイベントの 100 万個ごとに課金されます。

### Synthetics

* **API テスト**は、1 つの URL に対する 1 回の HTTP または HTTPS リクエストです。Datadog Synthetics サービスに対して実行された API テストランの 1 万回ごとに課金されます。
* **ブラウザテスト**は、仮想 Web ブラウザを使用して Web ベースアプリケーションに対してスクリプトによって行われる一連のユーザーアクションのシミュレーションです。Datadog Synthetics サービスに対して実行されたブラウザテストの 1,000 回ごとに課金されます。
