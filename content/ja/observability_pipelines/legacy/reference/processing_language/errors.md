Datadog Processing Language (DPL)、または Vector Remap Language (VRL) はフェイルセーフ言語です。つまり、潜在的なエラーがすべて処理されない限り、DPL/VRL プログラムはコンパイルされません。これにより、 DPL/VRL プログラムが不正なデータを処理できるようになります。
