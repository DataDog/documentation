## すべての Embed の取得
これまでに作成された埋め込み可能なグラフのリストを取得します。

戻り値: これまでに UI および API から作成された Embed に関する情報を含む JSON リスト。各 JSON グラフ応答は、GET api/v1/graph/embed/<EMBED_ID> から返される応答と同じ形式です。

##### 引数

このエンドポイントは、JSON 引数を受け取りません。
