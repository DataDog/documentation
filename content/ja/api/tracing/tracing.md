## トレーシング

詳細については、[Datadog を使用したトレーシング][1]および [APM の用語][2]のページを参照してください。

トレーシング API は、サービス側の API というよりはむしろ Agent の API です。トレースをローカルエンドポイント `http://localhost:8126/v0.3/traces` に送信して、それを Agent が Datadog に転送できるようにします。
