## Authentication
Datadog の API へのすべてのリクエストに認証が必要です。データ書き込みリクエストには、レポートアクセス権と `API キー`が必要です。データ読み取りリクエストには、フルアクセス権のほかに`アプリケーションキー`も必要です。

[アカウントの API とアプリケーションキーはこちらで管理できます][1]。

[1]: https://app.datadoghq.com/account/settings#api
