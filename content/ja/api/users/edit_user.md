## ユーザーを編集する

_この機能は、管理者に属するアプリケーションキーを持つ場合にのみ使用できます。_

ユーザーの属性を編集します。

**引数**:

* **`user_id`** [*必須*]: 編集するユーザーの ID。
* **`disabled`** [*任意*]: ユーザーを有効または無効にする。
* **`email`** [*任意*]: ユーザーに関連付ける電子メール。
* **`name`** [*任意*]: ユーザーに関連付ける名前。
