## ユーザーの更新
この機能は、管理者に属するアプリケーションキーを持つ場合にのみ使用できます。

##### 引数
* **`id`** [必須]:
    ユーザーのハンドル。
* **`name`** [オプション、デフォルト = **None**]:
    ユーザーの新しい名前。
* **`email`** [オプション、デフォルト = **None**]:
    ユーザーの新しい電子メールアドレス。
* **`disabled`** [オプション、デフォルト = **None**]:
    ユーザーの新しい無効ステータス。
* **`access_role`** [オプション、デフォルト = **st**]:
    ユーザーのアクセスロール。以下から選択します。
    *  **st** (標準ユーザー)
    *  **adm** (管理者ユーザー)
    *  **ro** (読み取り専用ユーザー)
