## メトリクスメタデータの編集

メトリクスメタデータエンドポイントを使用して、メトリクスのメタデータのフィールドを編集できます。
[サポートされているタイプについては、こちらを参照してください][1]。

**引数**:

* **`type`** [必須]:
    [メトリクスタイプ][1] (**gauge**、**rate** など)
* **`description`** [オプション、デフォルト = **None**]:
    メトリクスの説明の文字列
* **`short_name`** [必須]:
    メトリクスの短縮名の文字列
* **`unit`** [オプション、デフォルト = **None**]:
    メトリクスの主単位 (**byte**、**operation** など)
* **`per_unit`** [オプション、デフォルト = **None**]:
    メトリクスの基準単位。たとえば、**バイト/秒**の場合の**秒**
* **`statsd_interval`** [オプション、デフォルト = **None**]:
    該当する場合は、メトリクスの StatsD フラッシュ間隔 (秒単位)

[1]: /ja/developers/metrics
