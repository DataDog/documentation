Python Datadog Trace ライブラリはオープンソースです。詳細については、[Github リポジトリ][1]をご覧ください。

Python `2.7` および `3.4` バージョン以降がサポートされます。

## インテグレーション

ライブラリに関するサポートをご希望の場合は、[サポートチーム][2]までお気軽にお問い合わせください。
