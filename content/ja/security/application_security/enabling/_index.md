アプリケーション言語用の Datadog ライブラリを使用して、本番システムを標的とした[脅威の検出と保護][1]、およびコードとそのオープンソース依存関係における[リスク管理][2]を可能にします。サーバー、Docker、Kubernetes、AWS ECS、(対応言語の) AWS Fargate でホストされているアプリの脆弱性と脅威を検出することができます。
