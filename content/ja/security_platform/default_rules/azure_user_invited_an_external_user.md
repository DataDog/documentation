## 目標

外部ユーザーへ招待が送信されたことを検出します。

## 戦略

Azure Active Directory Audit ログを監視し、`@evt.name` が `Invite external user` と等しく、`@evt.outcome` が `success` と等しいことを検出します。

## トリアージと対応

1. 招待を確認し、その受信者と招待が有効であるかを判断します。
