## Description
Block scoped lexical declarations like `let` and `const` are preferred over `var`. Block scope is common in many other programming languages and helps programmers avoid mistakes.

## Compliant Code Examples
```javascript
const JOE = 'schmoe';
let moo = 'car';
const JOE = 'schmoe';
let moo = 'car';
```
