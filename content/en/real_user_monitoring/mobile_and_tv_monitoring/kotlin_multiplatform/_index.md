## Overview

Datadog Real User Monitoring (RUM) enables you to visualize and analyze the real-time performance and user journeys of your application's individual users.

The Datadog Kotlin Multiplatform SDK supports Android 5.0+ (API level 21) and iOS v12+.

## Start monitoring Kotlin Multiplatform applications

To get started with RUM for Kotlin Multiplatform, create an application and configure the Kotlin Multiplatform SDK.
