## Overview

{{< callout url="#" header="false" >}}
Quality Gates Rules are in beta.
{{< /callout >}}

Quality Gates Rules use GitHub Checks to block or unblock a CI/CD pipeline.

## Create a rule

Navigate to [**CI** > **Quality Gate Rules**][1] and click **+ New Rule**.

## Permissions

Enable the `Quality Gate Rules Write` permission for a user to create and edit Quality Gate Rules.
