## Description
Avoid sleeping in tests. It blocks the test execution and leads to unpredictable results.

## Non-Compliant Code Examples
```csharp
class MyClass {
    [TestMethod]
    public void TestMyMethod()
    {
        Thread.Sleep(500);
    }
}

```

## Compliant Code Examples
```csharp
class MyClass {
    [TestMethod]
    public void TestMyMethod()
    {
        // statements
    }
}

```
