## Metadata

**Language:** C#

**Severity:** Warning

**Category:** Error Prone

## Description
Do not throw exceptions in special methods such as `ToString()`, `Dispose()` or `GetHashCode`.

## Non-Compliant Code Examples
```csharp
class MyClass {
    public override string GetHashCode()
    {
        if (something)
        {
            throw new Exception();
        }
        throw new Exception();
    }
}

```

```csharp
class MyClass {
    public override string ToString()
    {
        if (something)
        {
            throw new Exception();
        }
        throw new Exception();
    }
}

```
