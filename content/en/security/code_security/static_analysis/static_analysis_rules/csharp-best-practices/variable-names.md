**Language:** C#

## Description
Avoid using language keywords to declare variables. While being authorized by the compiler, it makes the code harder to understand.

## Non-Compliant Code Examples
```csharp
using System.Net;

class MyClass {
    public static void routine()
    {
        bool await = false;
        bool async = true;
    }
}

```
