## Non-Compliant Code Examples
```kotlin
fun Foo() {}

fun Foo_Bar() {}

fun `Foo Bar`() {}

fun foo_bar() {}
```

## Compliant Code Examples
```kotlin
fun foo() {}

fun fooBar() {}

// Any keyword is allowed when wrapped between backticks
fun `fun`() {} 
```
