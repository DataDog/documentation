## Description
Operator `--` and `++` do not exists in Python. Increment or decrement the variable appropriately.

## Non-Compliant Code Examples
```python
--n  # use n = n - 1
++n  # use n = n + 1

```

## Compliant Code Examples
```python
n = n + 1
n = n - 1
```
