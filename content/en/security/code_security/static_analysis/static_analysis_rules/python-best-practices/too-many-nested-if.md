**Language:** Python

## Description
Too many nested loops make the code hard to read and understand. Simplify your code by removing nesting levels and separate code in small units.

## Non-Compliant Code Examples
```python
if foo < 0:
    if bar:
        if baz:
            if bao:
                pass
```

## Compliant Code Examples
```python
if foo:
    if bar:
        if baz:
            pass
```
