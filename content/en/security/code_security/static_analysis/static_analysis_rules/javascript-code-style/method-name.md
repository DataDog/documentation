## Description
Ensure that method names useÂ `camelCase` and not `snake_case` or `PascalCase`.

## Non-Compliant Code Examples
```javascript
const a = { GetValue() {} }
class A { set_value() {} }
class A { *set_value() {} }
class A { #set_value() {} }
class A { #SetValue() {} }
class A { _SetValue() {} }
class A { __Set_Value() {} }
```

## Compliant Code Examples
```javascript
const a = { getValue() {} }
class A { setValue() {} }
class A { #fooBla() {} }
class A { _setValue() {} }
class A { __setValue() {} }
class A { set2() {} }
```
