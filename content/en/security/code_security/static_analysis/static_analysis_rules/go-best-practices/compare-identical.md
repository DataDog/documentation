**Language:** Go

## Description
Avoid comparing the same expression, it always results to `true`.

## Non-Compliant Code Examples
```go
func main() {
    if a == a {

    }

    if a == b {

    }

    if (1 + 2) == (1 + 2) {

    }
}
```

## Compliant Code Examples
```go
func main() {
    if a == b {

    }
}
```
