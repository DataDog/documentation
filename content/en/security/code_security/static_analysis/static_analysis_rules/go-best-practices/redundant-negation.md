## Description
You do not need to convert the string into a slice of bytes to use `Write`, you can just use the string directly.

## Non-Compliant Code Examples
```go
package main

func main() {
	n := 5
	if !!(n == 5) {
		fmt.Println("hello")
	} else if !(!(n == 5)) {
		fmt.Println("hi")
	} 
	
	if !!true{
		fmt.Println("howdy")
	} else if !(!true){
		fmt.Println("partner")
	}

	fmt.Println("goodbye")
}

```

## Compliant Code Examples
```go
package main

func main() {
	n := 5
	if !(n == 5) {
		fmt.Println("hello")
	}

	fmt.Println("goodbye")
}

```
