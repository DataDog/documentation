API Security in Datadog [App and API Protection][1] (AAP) helps you discover API endpoints, understand the risk they expose, and verify how they behave.

API Security Inventory provides the catalog and risk view: it lists the APIs in your environment, the services that own them, and the findings tied to each. Endpoint Scanning enriches the inventory by actively probing eligible endpoints to confirm whether they are publicly reachable and whether they require authentication.
