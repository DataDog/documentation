API Security in Datadog App and API Protection (AAP) helps you discover API endpoints, understand endpoint risk, and verify endpoint behavior.

Use this section to find documentation for API Security capabilities:

- [API Security Inventory][1] provides a catalog of API endpoints, services, and findings across your environment.
- [Endpoint Scanning][2] enriches eligible endpoints with verified reachability and authentication data.
