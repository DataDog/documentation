This guide explains how to set up App and API Protection (AAP) for Ruby applications running in Docker containers. The setup involves:

1. Running the Datadog Agent
2. Configuring your Ruby application container
3. Enabling AAP monitoring

## Prerequisites

- Docker installed on your host
- Ruby application containerized with Docker
- Datadog Agent running in a container or is installed on the host

## 1. Running the Datadog Agent

Run the Datadog Agent in a docker container by following the [setup instructions for Docker](/agent/?tab=cloud_and_container).
