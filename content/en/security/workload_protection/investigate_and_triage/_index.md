Use the explorers and in-app views in Workload Protection to investigate runtime activity, triage threats, and review security posture findings.

## Agent events

[Agent events][1] are the raw telemetry generated by the Datadog Agent when runtime activity matches an agent rule. Use the Agent Events explorer to investigate events before they become signals.

## Signals

[Signals][2] are generated when agent events match a backend detection rule. Use the Signals Explorer to investigate threats, triage signals, and take response actions.

[Findings][3] are generated when agent events match a finding rule. Use the Findings explorer to review runtime security posture issues.
