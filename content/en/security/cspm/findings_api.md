Query and filter all findings like you would in the explorer, or get the full details for a specific finding via API.

## Endpoints

- **Base URL**: `https://app.datadoghq.com/api/unstable`
- **Findings list endpoint**: `GET /posture_management/findings`
- **Finding details**: `GET /posture_management/findings/{id}`
