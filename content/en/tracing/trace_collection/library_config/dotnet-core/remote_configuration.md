### Remote Configuration

`DD_REMOTE_CONFIGURATION_ENABLED`
: **Aliases**: `DD_REMOTE_CONFIG_ENABLED`<br>
**Type**: `boolean`<br>
**Default**: `true`<br>
Enable or disable remote configuration. Also accepts the alias `DD_REMOTE_CONFIG_ENABLED`.

`DD_REMOTE_CONFIG_POLL_INTERVAL_SECONDS`
: **Aliases**: `DD_INTERNAL_RCM_POLL_INTERVAL`<br>
**Type**: `decimal`<br>
**Default**: `5.0`<br>
Remote configuration polling interval in seconds.
