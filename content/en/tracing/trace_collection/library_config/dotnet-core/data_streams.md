### Data Streams

`DD_DATA_STREAMS_ENABLED`
: **Type**: `boolean`<br>
**Default**: `true`<br>
Enables data streams monitoring support.

`DD_DATA_STREAMS_LEGACY_HEADERS`
: **Type**: `boolean`<br>
**Default**: `false`<br>
Enables legacy binary headers in Data Streams Monitoring. False by default if DSM is in default state, true otherwise.
