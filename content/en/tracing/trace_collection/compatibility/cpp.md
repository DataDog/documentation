## Compatibility

The C++ Datadog Trace library is open source - view the [GitHub repository][1] for more information.

This library requires C++17 compiler to build.

Supported platforms include:
- `x86_64` and `arm64` Linux.
- `x86_64` Windows.
- `arm64` macOS.
