## Update host tags

This endpoint allows you to update/replace all tags in an integration source with those supplied in the request.

**ARGUMENTS**:

* **`tags`** [*required*]:
    A list of tags
* **`source`** [*optional*, *default*=**users**]:
    The source of the tags (e.g. chef, puppet).
