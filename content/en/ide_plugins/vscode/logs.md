## Overview

Use **Logs** to gauge the volume of logs generated by a given log line in your code. The extension adds annotations above your code to detected logging patterns that match log records in Datadog. Click an annotation to open the [Log Explorer][1] in Datadog and view the matching logs.

{{< img src="/developers/ide_plugins/vscode/logs_navigation.mp4" alt="Preview of Logs Navigation" style="width:100%" video=true >}}

You can also search Datadog logs from within the IDE. Select any text in the code editor, then right-click and select **Datadog \> Search Logs With Selected Text**.
