## Overview

{{< img src="observability_pipelines/packs/aws_alb.png" alt="The AWS Application Load Balancer Logs pack" style="width:25%;" >}}

AWS ALBs capture HTTP request activity routed through them.

What this pack does:

- Parses access logs into structured fields
- Extracts latency, status, and target metrics
- Normalizes request data
