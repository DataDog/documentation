## Overview

{{< img src="observability_pipelines/packs/zscaler_web_logs.png" alt="The Zscaler ZIA Web Logs pack" style="width:25%;" >}}

Zscaler Internet Access (ZIA) Web Logs capture user web activity and security actions.

What this pack does:

- Normalizes fields and adds threat context
- Drops routine and low-signal traffic
- Keeps risky, blocked, and unknown traffic
