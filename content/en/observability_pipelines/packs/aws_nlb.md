## Overview

{{< img src="observability_pipelines/packs/aws_nlb.png" alt="The AWS Network Load Balancer Logs pack" style="width:25%;" >}}

AWS NLBs captures connection activity and performance data.

What this pack does:

- Parses NLB access logs into structured fields
- Extracts connection, handshake, and traffic metrics
- Cleans and normalizes connection data
