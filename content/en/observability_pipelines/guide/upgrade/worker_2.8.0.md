Upgrade to Worker version 2.8.0 to have access to the following new features, enhancements, and fixes.

## New features

- Custom environment variable support for all sources and destinations.

## Enhancements

- Support data streams in the Elasticsearch destination as an indexing strategy.
- Template syntax support for the HTTP Client destination.

## Fixes

- Fixes for TLS enablement in the HTTP Server source.
- Fixes for worker health metrics.
- A fix for basic authentication in OpenSearch.
