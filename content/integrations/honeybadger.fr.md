## Aperçu

Connecter Honeybadger à Datadog pour:

* Consulter les erreurs dans le flux en temps réel
* Chercher vos erreurs sur vos graphiques
* Discuter des erreurs avec votre équipe
* Etre super génial

## Implémentation
### Installation

Pour capturer les erreurs depuis Honeybadger:

1.  Ouvrez votre [project list](https://app.honeybadger.io/users/sign_in) Honeybadger.

2.  Cliquez sur "Settings" pour le projet que vous voulez monitorer.

3.  Cliquez sur "Alerts & Integrations".

4.  Sélectionnez "Datadog" comme nouvelle intégration.

5.  Ajoutez votre clés d'[API](https://app.datadoghq.com/account/settings#api).

6.  Sauvegardez l'intégration.

7.  Cliquez sur le bouton **Install Integration** dans la [fenêtre d'intégration Honeybadger](https://app.datadoghq.com/account/settings#integrations/honeybadger).
