Overview

Connect NGiNX to Datadog in order to:
  • Visualize your web server performance
  • Correlate the performance of Nginx with the rest of your applications
From the open-source Agent: * NGiNX YAML Example * NGiNX checks.d The following metrics are collected by default with the NGiNX integration: nginx.net.connections nginx.net.reading nginx.net.request_per_s nginx.net.waiting nginx.net.writing
