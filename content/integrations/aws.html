--- title: Datadog-AWS Cloudwatch Integration --- The recommended way to configure Cloudwatch in Datadog is to create a new user via the Amazon [IAM Console](https://console.aws.amazon.com/iam/home#s=Home) and grant that user (or group of user) Amazon EC2 and Cloudwatch *read-only* access. These can be set via policy templates in the console.
