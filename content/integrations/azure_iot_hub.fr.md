## Aperçu
Azure IOT Hub est un service entièrement géré qui permet des communications fiables, sécurisés et bidirectionnelle entre des millions de périphériques IoT.

Obtenir les métriques de Azure IOT Hub pour:

* Visualiser les performances de vos hubs IoT
* Corréler les performances de vos hubs IoT avec vos applications
