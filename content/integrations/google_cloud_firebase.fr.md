## Aperçu
Firebase est une plateforme mobile qui vous aide à développer rapidement des applications, étendre votre base de donnée utilisateur, et faire plus d'argent.

Obtenir les métriques de Google Firebase pour:

* Visualiser les performances de votre base de données Firebase et de vos services hébergés.
* Corréler les performances de vos outils Firebase avec vos applications
