Monitors

Monitors allow you to watch a metric or check that you care about and notifies your team when a defined threshold has exceeded.

For more information, see Creating Monitors.

Note: curl commands require url encoding.

  • POST Create a monitor
  • GET Get all monitors
  • PUT Edit a monitor
  • GET Get a monitor's details
  • DELETE Delete a monitor
  • GET Check if a monitor can be deleted
  • POST Validate a monitor
  • POST Validate an existing monitor
  • GET Get a monitor configuration policy
  • GET Monitors search
  • GET Get all monitor configuration policies
  • GET Monitors group search
  • POST Create a monitor configuration policy
  • POST Unmute a monitor
  • PATCH Edit a monitor configuration policy
  • POST Mute a monitor
  • DELETE Delete a monitor configuration policy
  • POST Unmute all monitors
  • GET Get a monitor notification rule
  • POST Mute all monitors
  • GET Get all monitor notification rules
  • POST Create a monitor notification rule
  • PATCH Update a monitor notification rule
  • DELETE Delete a monitor notification rule
  • GET Get a monitor user template
  • GET Get all monitor user templates
  • POST Create a monitor user template
  • PUT Update a monitor user template to a new version
  • DELETE Delete a monitor user template
  • POST Validate a monitor user template
  • POST Validate an existing monitor user template
