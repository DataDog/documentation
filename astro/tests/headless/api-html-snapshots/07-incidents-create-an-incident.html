Create an incident

WARNING Note: This endpoint is in public beta. If you have any feedback, contact Datadog support.
POST https://api.datadoghq.com/api/v2/incidentshttps://api.us3.datadoghq.com/api/v2/incidentshttps://api.us5.datadoghq.com/api/v2/incidentshttps://api.datadoghq.eu/api/v2/incidentshttps://api.ap1.datadoghq.com/api/v2/incidentshttps://api.ap2.datadoghq.com/api/v2/incidentshttps://api.ddog-gov.com/api/v2/incidents

Overview

Create an incident.

Permissions: incident_write

OAuth apps require the incident_write authorization scope to access this endpoint.

Request Body (required)

Incident payload.
